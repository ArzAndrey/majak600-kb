# HMI: Menu — горизонтальное и вертикальное меню

## Зависимости
- HMI/01_SYNTAX_BASE.md — базовый синтаксис
- CONFIG/03_COLORS.md — цветовые константы

---

## Общее описание

На экране всегда присутствуют два ряда меню:
- **Горизонтальный ряд** — клавиши F1–F6
- **Вертикальный ряд** — клавиши F7–F14

Специальные клавиши для расширения меню:
- **Ctrl** — альтернативный набор функций (ctrlF1–ctrlF6, ctrlF7–ctrlF14)
- **Alt** — альтернативный набор функций (altF1–altF6, altF7–altF14)

Смены меню происходят независимо от остальных зон экрана.

---

## Синтаксис

{ menu номер_меню
    { f1 (access) { text "Текст" (параметры) { условие } } { pressed команда } { released команда } }
    { f2 ... }
    ...
    { f6 ... }
    { f7 ... }
    { f8 ... }
    ...
    { f14 ... }
} menu

### Горизонтальные клавиши (F1–F6)
- f1, f2, f3, f4, f5, f6 — описание клавиш горизонтального меню
- ctrlf1 ... ctrlf6 — при нажатой клавише Ctrl
- altf1 ... altf6 — при нажатой клавише Alt

### Вертикальные клавиши (F7–F14)
- f7, f8, f9, f10, f11, f12, f13, f14 — описание клавиш вертикального меню
- ctrlf7 ... ctrlf14 — при нажатой клавише Ctrl
- altf7 ... altf14 — при нажатой клавише Alt

Если клавиша не используется, её можно пропустить.

## Параметры клавиши

### access (уровень доступа)
Необязательный параметр. Определяет, при каком уровне доступа клавиша будет доступна.

Пример:
(access=-1) — клавиша доступна только при отрицательном уровне (специальные права)

### text "Текст"
Текст, выводимый на экране для обозначения клавиши.
Для разбиения текста на строки используется символ $:
"Запись$в файл"

В описании клавиши может быть несколько блоков text — для разных состояний.

### Параметры text (необязательные)
| Параметр | Описание |
|----------|----------|
| fgr | Цвет символов |
| bgr | Цвет фона клавиши |

Пример:
{ text "Пуск записи$" { text "1" (fgr=CPARAM3) } }

### Условие в text
Работает аналогично условию в item, но без format.

Пример:
{ text "вкл" (fgr=RGB(150,150,150)) { if (d154#1) fgr=RGB(255,255,0) } }

### pressed команда
Действие, выполняемое при нажатии клавиши.

### released команда (необязательный)
Действие, выполняемое при отжатии клавиши.

## Команды pressed / released

### Переключение меню
| Команда | Описание |
|---------|----------|
| mm | Переключение на предыдущее меню |
| mmN | Переключение на указанное меню (N — номер) |
| mm10 | Переключение на меню 10 |

### Вызов сетки или модуля
| Команда | Описание |
|---------|----------|
| pressed имя_сетки | Вызов сетки |
| pressed имя_модуля | Вызов модуля |

### Команды с условием (ifok)
Синтаксис:
%=условие ifok число_команд команда_1 0 команда_2

- условие — выражение сравнения
- число_команд — количество команд (команда_1 + команда_2)
- команда_1 — выполняется, если условие ложно
- команда_2 — выполняется, если условие истинно

Пример:
%=d404#0 ifok 3 "1" start_log 0

### Команды с условием (ifnotok)
Синтаксис:
%=условие ifnotok число_команд команда_1 0 команда_2

- команда_1 — выполняется, если условие истинно
- команда_2 — выполняется, если условие ложно

### Другие команды
| Команда | Описание |
|---------|----------|
| inputkill | Очистить наборную строку |
| задание D-ячейки | d130=3 |
| задание параметра | d130=3 |

## Примеры меню

### Пример 1: Простое меню с переключением

{ menu 20 ; Режим Ручной
    { f1 { text "Шпиндель" } { pressed mm201 } }
    { f2 { text "СОЖ" } { pressed mm202 } }
    { f3 { text "Сброс" } { pressed s34 } }
    { f6 { text "Возврат" } { pressed mm } }
} menu

### Пример 2: Меню с условиями и текстом на несколько строк

{ menu 201 ; Управление шпинделем
    { f1 { text "M3" } { pressed d130=3 } }
    { f2 { text "M4" } { pressed d130=4 } }
    { f3 { text "M5" } { pressed d130=5 } }
    { f6 { text "Возврат" } { pressed mm } }
} menu

### Пример 3: Меню с изменением цвета в зависимости от состояния

{ menu 20 ; Режим Ручной
    { f2
        { text "СОЖ $" }
        { text " вкл " (fgr=RGB(150,150,150)) { if (d154#1) fgr=RGB(255,255,0) } }
        { text " выкл " (fgr=RGB(150,150,150)) { if (d154=1) fgr=RGB(255,255,0) } }
        { pressed %=d154#1 ifok 2 d154=0 0 d154=1 }
    }
} menu

### Пример 4: Меню осциллографа (с использованием Ctrl)

{ menu 1070 ; oscilloscope - просмотр сигнала
    { f1
        { text "Просмотр" }
        { text "$сигнал" { fgr =CPARAM} }
        { text "$память" { fgr =CPARAM3} }
        { text " файл" { fgr =CPARAM3} }
        { pressed view_oscilloscope_buffer mm1071 }
    }
    { Ctrl6
        { text "Возврат" }
        { pressed next }
    }
    { f7
        { text "Копировать$память$в файл 1" }
        { pressed "1" save_buffers }
    }
} menu

### Пример 5: Меню с условием ifok

{ menu 1070
    { f8
        { text "Пуск записи$" }
        { text "1" (fgr=CPARAM3) { if d404=0 fgr=MENU_FGR } }
        { text " 2" (fgr=CPARAM3) { if d404=1 fgr=MENU_FGR } }
        { text " 3" (fgr=CPARAM3) { if d404=2 fgr=MENU_FGR } }
        { pressed %=d404#0 ifok 3 "1" start_log 0
                  %=d404#1 ifok 3 "2" start_log 0
                  %=d404#2 ifok 3 "3" start_log 0 }
    }
} menu

## Меню переключения режимов

Специальное меню (номер 0), вызываемое клавишей РЕЖИМ (Tab). Использует только горизонтальный ряд.

{ menu 0 ; tab-menu
    { f1 { text "Ручной" } { pressed !s1-0 form mm } }
    { f2 { text "Автомат" } { pressed !s0-0 form mm } }
    { f3 { text "Выход$в ноль" } { pressed !s2-0 form mm } }
    { f4 { text "Выход$в точку" } { pressed !s6-0 form mm } }
    { f5 { text "Фиксир.$перемещ."} { pressed !s8-0 form mm d406=1 } }
    { f6 { text "Преднабор" } { pressed !s7-0 form mm } }
    { altf3 (access=-1) { text "Выполнить" } { pressed execute } }
} menu

---

## Правила оформления

1. Каждое меню начинается с { menu номер_меню и заканчивается } menu
2. Клавиши описываются в порядке возрастания номеров (пропущенные клавиши игнорируются)
3. Текст на клавише может быть многострочным (разделитель $)
4. Цвет текста и фона могут меняться по условию
5. Команды pressed и released выполняют действия при нажатии/отжатии

{ menu 100

        { text "$подачи" }
        { pressed mm101 }
    }

        { text "Выход" }

    }
} menu
