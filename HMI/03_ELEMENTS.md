## Item (элемент)

Команда индикации строки (вывод текста или значения переменной).

### Синтаксис
{ item имя_элемента (параметры) координата_x координата_y инверсия показывать_всегда 'текст' команды {условие} }

### Имя элемента
- `#` — пустое выражение (выводится только текст)
- `d70` — имя D-ячейки (выводится значение)
- `m30` — имя массива (выводится значение)
- `u12` — имя параметра (выводится значение)
- `time` — часы
- `up` — имя файла с УП
- `comment` — комментарий к УП
- `edit_name` — имя редактируемого файла
- `txted_pos` — позиция курсора в тексте
- `txted_name` — название редактируемого файла
- `diskfree` — размер свободного дискового пространства
- `m32.0`, `m32.1` ... — названия осей из справочника

### Параметры элемента

| Параметр | Описание | Пример |
|----------|----------|--------|
| height | Высота строки (кегль) в пикселях | height=20 |
| width | Ширина строки в пикселях (если не задана — авто) | width=100 |
| fgr | Цвет текста | fgr=MAIN_TEXT |
| bgr | Цвет фона | bgr=MAIN_BGR |
| invfgr | Цвет текста при инверсии | invfgr=MAIN_BGR |
| invbgr | Цвет фона при инверсии | invbgr=CHANNEL0 |
| align | Выравнивание текста | align=center |
| fonts | Гарнитура шрифта | fonts='IBMPlexMono' |

**Константы выравнивания:**
- `TEXT_LEFT` / `LEFT` — по левому краю
- `TEXT_RIGHT` / `RIGHT` — по правому краю
- `TEXT_CENTERED` / `CENTER` — по центру
- `LEFT+BOTTOM` — комбинация (лево + низ)

**Специальные шрифты:**
- `'Mayak-Icons'` — иконочный шрифт для вывода символов ('c', 't', 'S', 't' и др.)

### Координаты
Координаты задаются в пикселях относительно левого верхнего угла View.
Можно использовать числа или константы:
- `0`, `10`, `GAP_H`, `GAP_V`
- арифметические выражения: `BOX_X+BOX_GAP_H`

### Инверсия
- `yes` — инвертировать цвета (меняет fgr и bgr местами)
- `no` — без инверсии

### Показывать всегда
- `yes` — индицировать даже при нулевом значении переменной
- `no` — не индицировать при нулевом значении

### Текст

#### Простой текст
'Автомат'
'Скорость: мм/мин'

#### Вывод из массива m32 (названия осей)
Синтаксис `'&N'`, где N — номер оси (0-5):
'&0'   ; выводит название оси 0 (например, "X")
'&1'   ; выводит название оси 1 (например, "Y")
'&2'   ; выводит название оси 2 (например, "Z")

#### Форматированный вывод
| Формат | Описание | Пример | Результат |
|--------|----------|--------|----------|
| [ ] | Простое число | [ ] | 12345 |
| [m] | m знаков, дополнение нулями слева | [5] | 00123 |
| [+m] | m знаков с плюсом/минусом | [+5] | +0123 |
| [m.n] | m знаков всего, n после точки | [9.3] | 00012.345 |
| [+m.n] | с плюсом/минусом | [+9.3] | +0012.345 |
| [m!n] | отсечь n знаков справа | [6!2] | 00123 |
| &(выражение) | вычисление выражения | &(m28*1000/m29)[] | |
| <n> | текст из txtvals.dat по номеру n | <51> | Не задана зона... |
| <!n> | текст из txtvals.dat (альт. формат) | <!12> | 12:34:56 |

### Команды
Команды добавляют функциональность (изменение значения переменной при нажатии клавиш →/←).

Синтаксис:
`+aa#bb -cc#dd`

- `+aa#bb` — увеличение: aa — команда при нажатии →, bb — при отжатии
- `-cc#dd` — уменьшение: cc — команда при нажатии ←, dd — при отжатии

Команды описываются в файле action.dat:
{ action dpl d65 = 1 }
{ action dmn d65 = 2 }
{ action I10 d317=1 }
{ action N20 d317=0 }

Пример элемента с одной парой команд:
{ item d65 (height=30) 516 66 no yes 'Позиция [1]' +dpl -dmn }

Пример элемента с несколькими командами:
; В файле box_f_avt.dat
{ item d99 (height=20) 260 65 no yes '+ff1 -ff2 +df1 -df2' }
Здесь:
- `+ff1` и `-ff2` связаны со стрелками «→»/«←».
- `+df1` и `-df2` — дополнительные команды, возможно, для других действий.

#### Специальный синтаксис: `!"<имя_действия>"#"<значение>"`
Позволяет вызвать действие, передав ему значение, введенное пользователем в наборную строку (`<ust>`).
; Пример из box_fix.dat
{ item # (height=20) 260 65 no yes '!"<ust>PF"#"<ust>"' }
Здесь при нажатии на элемент будет вызвано действие `PF` (определенное в `action.dat`), и ему в качестве параметра будет передано текущее содержимое наборной строки (`<ust>`).

### Условие
Условия позволяют менять параметры элемента и текст в зависимости от значений переменных.

Синтаксис:
{ if выражение_условия новые_значения_параметров format='Текст' }

Выражения условия:
| Синтаксис | Описание |
|-----------|----------|
| m30=0 | Равно |
| m30#0 | Не равно |
| m30>10 | Больше |
| m30<5 | Меньше |
| (p7=0)&&(p30=p2) | И |
| (p7=1)||(p30=p8) | ИЛИ |
| p10&1#0 | Битовое И (проверка 1-го бита) |
| p10&2#0 | Битовое И (проверка 2-го бита) |
| p10&4#0 | Битовое И (проверка 3-го бита) |

Ссылки на ячейки БПрО:
| Тип | Синтаксис | Пример |
|-----|-----------|--------|
| D-ячейки | dномер | d70 |
| Параметры 0-299 | uномер | u12 |
| Параметры 300-399 | сномер | c124 |
| Параметры 400-499 | иномер.координата | u0.0 |
| Параметры 500-599 | сномер.координата | c15.1 |
| Массив M | mномер | m30 |
| Массив M с координатой | mномер.координата | m1.0 |
| Массив I | iномер | i2 |
| Массив I с координатой | iномер.координата | i6.1 |
| Массив P | pномер | p0 |

Пример:
{ item m30 (invbgr=Error_C invfgr=MAIN_BGR width=106)
    111 GAP_V yes yes 'ЧПУ: Авария'
    { if m30=255 invbgr=GREEN format='ЧПУ: Готово' }
}

### Примеры item

#### Простой вывод текста
{ item # (height=20) GAP_H GAP_V yes yes 'Автомат' }

#### Вывод значения переменной с форматированием
{ item d70 (height=20 width=50) 100 50 no yes '[5]' }

#### Вывод названия оси из m32
{ item m32.0 (height=20) 100 50 no yes '&0' }

#### Вывод с условием изменения цвета
{ item m20 (fgr=B3 width=160) 111 GAP_V no yes '<24>'
    { if m20=0 fgr=MAIN_BGR }
}

#### Вывод с командой изменения значения
{ item d65 (height=30) 516 66 no yes 'Позиция [1]' +dpl -dmn }

#### Вывод с несколькими командами
{ item d99 (height=20) 260 65 no yes '+ff1 -ff2 +df1 -df2' }

#### Вывод с передачей значения из наборной строки
{ item # (height=20) 260 65 no yes '!"<ust>PF"#"<ust>"' }

## Box (залитый прямоугольник)

Команда индикации залитого прямоугольника.

### Синтаксис
{ box color=цвет x=координата_x y=координата_y x2=ширина y2=высота }
{ box color=цвет ifcolors='цвет if условие; цвет if условие' x=... y=... x2=... y2=... }

### Параметры
| Параметр | Описание |
|----------|----------|
| color | Цвет заливки (константа или RGB) |
| ifcolors | Условное изменение цвета (несколько условий через `;`) |
| x | Координата X левого верхнего угла |
| y | Координата Y левого верхнего угла |
| x2 | Ширина в пикселях |
| y2 | Высота в пикселях |

### Условный цвет (ifcolors)
Позволяет менять цвет прямоугольника в зависимости от условия:
{ box color=BOX_ZL_AVT_BGR_LOAD
    ifcolors='BOX_ZL_AVT_FGR_LOAD if (d710>0)&&(IND_KRD0=1); BOX_ZL_AVT_FGR_NO_LOAD if IND_KRD0=0'
    x=... y=... x2=... y2=... }
Синтаксис: `цвет if условие; цвет if условие` — несколько условий, разделенных `;`.

### Пример
{ box color=RGB(255, 205, 0) x=506 y=112 x2=142 y2=84 }

## Rect (рамка прямоугольника)

Команда индикации рамки прямоугольника.

### Синтаксис
{ rect color=цвет x=координата_x y=координата_y x2=ширина y2=высота }
{ rect color=цвет ifcolors='цвет if условие' x=... y=... x2=... y2=... }

### Параметры
| Параметр | Описание |
|----------|----------|
| color | Цвет рамки (константа или RGB) |
| ifcolors | Условное изменение цвета |
| x | Координата X левого верхнего угла |
| y | Координата Y левого верхнего угла |
| x2 | Ширина в пикселях |
| y2 | Высота в пикселях |

### Пример
{ rect color=RGB(255, 205, 0) x=300 y=10 x2=50 y2=25 }

## Plugin (встраиваемый модуль)

Специальный блок для вставки встроенных плагинов (редактор УП, осциллограф и т.д.).

### Синтаксис
{ имя_плагина
    plugin_x = координата_X
    plugin_y = координата_Y
    plugin_w = ширина
    plugin_h = высота
    параметр1 = значение1
    параметр2 = значение2
}

### Пример (индикатор УП)
{ indicator_UP
    plugin_x = BOX_UP_X
    plugin_y = BOX_UP_Y+24
    plugin_w = BOX_UP_W*1.15
    plugin_h = BOX_UP_H-24-BOX_UP_HEIGHT
    max_row = 16-2*IND_KRD0-2*IND_KRD1-2*IND_KRD2-2*IND_KRD3-2*IND_KRD4-2*IND_KRD5
    max_col = 50
}

## Пример комплексного использования

{ view (xmax=800 ymax=600 windowed=1 bgr=MAIN_BGR)

    ; Прямоугольник-рамка
    { rect color=MAIN_TEXT x=10 y=10 x2=780 y2=580 }

    ; Заголовок
    { item # (height=FONTSIZE_BIG fgr=AN_C) GAP_H GAP_V yes yes 'Режим: Автомат' }

    ; Блок с координатами
    { box color=REG_F x=BOX_X y=BOX_F_Y x2=BOX_W y2=BOX_F_H }
    { item # (height=FONTSIZE_MID) BOX_X+BOX_GAP_H BOX_F_Y+BOX_GAP_V no yes 'F' }

    ; Значение переменной с условием
    { item m30 (width=106)
        111 GAP_V yes yes 'ЧПУ: Авария'
        { if m30=255 invbgr=GREEN format='ЧПУ: Готово' }
    }

} view
