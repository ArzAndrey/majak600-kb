# HMI: Форматы вывода чисел и текста

## Зависимости
- HMI/01_SYNTAX_BASE.md — базовый синтаксис
- HMI/03_ELEMENTS.md — элемент item

---

## Общее описание

В элементе item формат вывода задаётся внутри кавычек после параметров. Формат определяет, как будет отображаться значение переменной или вычисленного выражения.

---

## Форматы чисел

### [ ] — простое число без форматирования
Выводит число как есть, без дополнения нулями.

| Пример | Значение | Вывод |
|--------|----------|-------|
| [ ] | 12345 | 12345 |
| [ ] | -12 | -12 |

### [m] — фиксированная ширина, дополнение нулями слева
m — общее количество знаков (включая число). Применяется только для положительных чисел.

| Пример | Значение | Вывод |
|--------|----------|-------|
| [5] | 12345 | 12345 |
| [5] | 12 | 00012 |
| [5] | 0 | 00000 |

### [+m] — фиксированная ширина с плюсом/минусом
m — общее количество знаков, включая знак.

| Пример | Значение | Вывод |
|--------|----------|-------|
| [+7] | 12345 | +012345 |
| [+7] | -12345 | -012345 |
| [+7] | 0 | +000000 |

### [m.n] — фиксированная ширина с десятичной точкой
m — общее количество знаков (включая точку и цифры после неё)
n — количество знаков после точки
Применяется только для положительных чисел.

| Пример | Значение | Вывод |
|--------|----------|-------|
| [9.3] | 12345 | 00012.345 |
| [9.1] | 12345 | 0001234.5 |
| [7.2] | 12 | 00012.00 |

### [+m.n] — фиксированная ширина с плюсом/минусом и десятичной точкой
m — общее количество знаков (включая знак, точку и цифры после неё)
n — количество знаков после точки

| Пример | Значение | Вывод |
|--------|----------|-------|
| [+9.3] | 12345 | +0012.345 |
| [+9.3] | -12345 | -0012.345 |
| [+9.3] | 0 | +0000.000 |

### [m!n] — отсечение n знаков справа
m — общее количество знаков
n — количество знаков, отсекаемых с конца
Применяется только для положительных чисел.

| Пример | Значение | Вывод |
|--------|----------|-------|
| [6!2] | 12345 | 00123 |
| [6!1] | 12345 | 01234 |

### [+m!n] — отсечение n знаков справа с плюсом/минусом
m — общее количество знаков (включая знак)
n — количество знаков, отсекаемых с конца

| Пример | Значение | Вывод |
|--------|----------|-------|
| [+6!2] | 12345 | +00123 |
| [+6!2] | -12345 | -00123 |

## Вычисляемые выражения &()

### Синтаксис
&(выражение)[формат]

Выражение может содержать:
- числа
- имена переменных (d70, m30, u12 и т.д.)
- арифметические операции (+, -, *, /)

### Примеры
| Формат | Значение | Вывод |
|--------|----------|-------|
| &(-568)[] | -568 | -568 |
| &(m28*1000/m29+55)[10.3] | m28=172, m29=10 | 000017.255 |

### Особенности
- Всегда выводится число или результат вычислений
- Игнорируется имя элемента (если указано d70, будет выведено &(...))

## Тексты из файла txtvals.dat <n>

### Синтаксис
<n> — где n — номер текстовой строки в файле txtvals.dat

### Назначение
Позволяет выводить предопределённые текстовые сообщения (ошибки, предупреждения, системные сообщения).

### Пример
| Формат | Значение d38 | Вывод |
|--------|--------------|-------|
| <51> | 51 | Не задана зона рассогласования при позиционировании |
| <25> | 25 | Не задана зона рассогласос |

### Обрезание текста
Если текст длиннее отведённого места, он обрезается по количеству символов, указанному в формате.

## Комбинирование форматов

Форматы можно комбинировать в одной строке.

### Синтаксис
'Текст [формат] текст <n> текст &(выражение)[формат]'

### Пример
| Формат | Значение | Вывод |
|--------|----------|-------|
| '![3] <51>' | d38=172 | !172 Не задана зона рассогласования при позиционировании |
| 'Скорость: [5] мм/мин' | d40=240 | Скорость: 00240 мм/мин |
| 'Ошибка: <25>' | d38=25 | Ошибка: Не задана зона рассогласос |

---

## Примеры использования в item

### Простой вывод числа
{ item d70 (height=20) 100 50 no yes '[5]' }

### Вывод числа с текстом
{ item d40 (height=20) 100 50 no yes 'Скорость: [5] мм/мин' }

### Вывод с плюсом/минусом
{ item d65 (height=20) 100 50 no yes '[+7]' }

### Вывод с десятичной точкой
{ item d78 (height=20) 100 50 no yes 'Перемещение: [9.3] мм' }

### Вывод вычисляемого значения
{ item # (height=20) 100 50 no yes '&(d78/1000)[6.3] мм' }

### Вывод текстового сообщения из txtvals.dat
{ item d38 (height=20) 100 50 no yes '<51>' }

### Комбинированный вывод
{ item d38 (height=20) 100 50 no yes 'Ошибка: <51> [3]' }

## Полная таблица форматов

| Формат | Описание | Знак | Десятичная точка | Обрезание справа |
|--------|----------|------|------------------|------------------|
| [ ] | Простое число | + | нет | нет |
| [m] | Фикс. ширина | нет | нет | нет |
| [+m] | Фикс. ширина со знаком | + | нет | нет |
| [m.n] | Фикс. ширина с точкой | нет | + | нет |
| [+m.n] | Фикс. ширина со знаком и точкой | + | + | нет |
| [m!n] | Отсечение справа | нет | нет | + |
| [+m!n] | Отсечение справа со знаком | + | нет | + |
| &(выр)[] | Вычисляемое выражение | + | опционально | опционально |
| <n> | Текст из txtvals.dat | нет | нет | нет |

---

## Особенности

1. **Форматы [m], [m.n], [m!n] работают только с положительными числами**
   Если значение отрицательное, формат может работать некорректно.

2. **Форматы [+m], [+m.n], [+m!n] работают с любыми числами**
   Автоматически добавляют знак + или -.

3. **При комбинировании форматов порядок не важен**
   'Текст [5] <51> &(d70)[3]'

4. **Файл txtvals.dat**
   Содержит предопределённые текстовые сообщения. Номера сообщений соответствуют кодам ошибок и системным сообщениям.

5. **Вычисляемые выражения**
   Могут быть сложными, включать несколько переменных и операции.

## Пример комплексного item с форматированием

{ item d78 (height=FONTSIZE_MID width=150 fgr=MAIN_TEXT)
    100 50 no yes
    'Величина: [9.3] мм  (&(d78/25.4)[6.2] дюйм)'
}

Вывод при d78=12345: Величина: 00012.345 мм  (000486.02 дюйм)
