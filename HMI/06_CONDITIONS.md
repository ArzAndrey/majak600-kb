## Общее описание

Условия позволяют динамически изменять:
- Текст элемента (item)
- Цвет текста и фона
- Параметры отображения
- Поведение меню

Условия применяются в:
- Элементах item (в конце конструкции)
- Меню (внутри text)
- Графических примитивах (box, rect, line)

---

## Синтаксис условия в item

{ if выражение_условия новые_значения_параметров format='Текст' }

Условие находится в конце элемента и всегда заключено в фигурные скобки {}.

### Структура
1. if — ключевое слово
2. выражение_условия — логическое выражение
3. новые_значения_параметров — параметры, применяемые при истинности условия
4. format='Текст' — текст, выводимый при истинности условия (необязательно)

---

## Выражения условия

### Синтаксис
переменная знак_сравнения значение

### Переменные
| Тип | Синтаксис | Пример |
|-----|-----------|--------|
| D-ячейки | dномер | d70 |
| Параметры 0-299 | uномер | u12 |
| Параметры 300-399 | сномер | c124 |
| Параметры 400-499 | иномер.координата | u0.0 |
| Параметры 500-599 | сномер.координата | c15.1 |
| Массив M | mномер | m30 |
| Массив M с координатой | mномер.координата | m1.0 |
| Массив I | iномер | i2 |
| Массив I с координатой | iномер.координата | i6.1 |
| Массив P | pномер | p0 |

### Знаки сравнения
| Знак | Описание |
|------|----------|
| = | Равно |
| # | Не равно |
| > | Больше |
| < | Меньше |
| >= | Больше или равно |
| <= | Меньше или равно |

### Битовые операторы
| Оператор | Описание | Пример |
|----------|----------|--------|
| & | Побитовое И | p10&1#0 (проверка 1-го бита) |
| | | Побитовое ИЛИ | p10|2#0 |

### Примеры выражений
| Выражение | Описание |
|-----------|----------|
| m30=0 | m30 равно 0 |
| m30#0 | m30 не равно 0 |
| m30=255 | m30 равно 255 |
| d70>100 | d70 больше 100 |
| u12<50 | параметр 12 меньше 50 |
| p7=0 | p7 равно 0 |
| p10&1#0 | проверка 1-го бита p10 |
| p11&2#0 | проверка 2-го бита p11 |

## Комбинирование условий

### Логические операторы
| Оператор | Описание |
|----------|----------|
| && | И (AND) |
| \|\| | ИЛИ (OR) |

### Синтаксис
(выражение1) && (выражение2)
(выражение1) || (выражение2)

### Примеры
| Условие | Описание |
|---------|----------|
| (p7=0)&&(p30=p2) | p7=0 И p30=p2 |
| (p7=1)||(p30=p8) | p7=1 ИЛИ p30=p8 |
| ((p7=0)&&(p30=p2))||((p7=1)&&(p30=p8)) | Сложная комбинация |
| ((u195=1)&&(m11=95)&&(m6#0)&&(m6#4)&&(m6#28)&&(u294=0)) | Множественное И |

## Формат 'Текст'

При выполнении условия можно заменить выводимый текст.

### Синтаксис
format='Новый текст'

### Пример
format='ЧПУ: Готово'

## Примеры условий в item

### Пример 1: Изменение текста
{ item m30 (invbgr=Error_C invfgr=MAIN_BGR width=106)
    111 GAP_V yes yes 'ЧПУ: Авария'
    { if m30=255 invbgr=GREEN format='ЧПУ: Готово' }
}

### Пример 2: Изменение цвета
{ item m20 (fgr=B3 width=160)
    111 GAP_V no yes '<24>'
    { if m20=0 fgr=MAIN_BGR }
}

### Пример 3: Изменение нескольких параметров
{ item d70 (height=20 fgr=MAIN_TEXT bgr=MAIN_BGR)
    100 50 no yes '[5]'
    { if d70>100 fgr=RED bgr=YELLOW }
    { if d70<0 fgr=BLUE format='Ошибка' }
}

### Пример 4: Сложное условие
{ item d38 (height=20)
    100 50 no yes 'Статус: [3]'
    { if (d38=0)||(d38=255) fgr=GREEN }
    { if (d38>0)&&(d38<255) fgr=RED }
}

### Пример 5: Множественное условие (из реального проекта)
{ item d317 (height=20) 260 65 no yes '+I10#StopNt -I20#StopNt'
    { if ((u195=1)&&(m11=95)&&(m6#0)&&(m6#4)&&(m6#28)&&(u294=0)) format='[6.3]' }
}

### Пример 6: Условие с битовой операцией
{ item # (height=20) 100 50 no yes ''
    { if (m0=6)&&(IND_KRD0=1)&&(p10&1#0) format='=' }
    { if (m0#6)&&(IND_KRD0=1)&&(p11&1#0) format='*' }
}

## Примеры условий в меню

### Пример 1: Изменение цвета текста по условию
{ menu 20
    { f2
        { text "СОЖ $" }
        { text " вкл " (fgr=RGB(150,150,150)) { if (d154#1) fgr=RGB(255,255,0) } }
        { text " выкл " (fgr=RGB(150,150,150)) { if (d154=1) fgr=RGB(255,255,0) } }
    }
} menu

### Пример 2: Условное отображение текста
{ menu 1070
    { f8
        { text "Пуск записи$" }
        { text "1" (fgr=CPARAM3) { if d404=0 fgr=MENU_FGR } }
        { text " 2" (fgr=CPARAM3) { if d404=1 fgr=MENU_FGR } }
        { text " 3" (fgr=CPARAM3) { if d404=2 fgr=MENU_FGR } }
    }
} menu

### Пример 3: Условное отображение текста через #номер_варианта
{ menu 20
    { f1
        { text "#1" (fgr=RGB(150,150,150)) { if d311=0 fgr=RGB(255,255,0) } }
        { text "вкл" (fgr=RGB(150,150,150)) { if d311=1 fgr=RGB(255,255,0) } }
        { pressed %=d311#1 ifok 3 d311=1 0 d311=0 }
    }
} menu

## Примеры условий в графических примитивах

### Box с условным цветом
{ box color=MAIN_BGR
    ifcolors='GREEN if m30=255'
    x=100 y=100 x2=200 y2=50
}

### Box с несколькими условиями
{ box color=BOX_ZL_AVT_BGR_LOAD
    ifcolors='BOX_ZL_AVT_FGR_LOAD if (d710>0)&&(IND_KRD0=1); BOX_ZL_AVT_FGR_NO_LOAD if IND_KRD0=0'
    x=100 y=100 x2=200 y2=50
}

### Rect с условным цветом
{ rect color=MAIN_TEXT
    ifcolors='RED if d70>100'
    x=100 y=100 x2=200 y2=50
}

### Line с условным цветом
{ line color=MAIN_TEXT
    ifcolors='YELLOW if m20=0'
    x=100 y=100 x2=200 y2=150
}

## Полный пример с несколькими условиями

{ view (xmax=800 ymax=600 bgr=MAIN_BGR)
    
    ; Заголовок с изменяемым цветом
    { item # (height=FONTSIZE_BIG)
        GAP_H GAP_V yes yes 'Статус системы'
        { if m30=0 fgr=YELLOW }
        { if m30=255 fgr=GREEN }
        { if (m30>0)&&(m30<255) fgr=RED }
    }
    
    ; Значение с условным форматированием
    { item d70 (height=FONTSIZE_MID width=100)
        GAP_H GAP_V+HEIGHT_MID no yes '[5]'
        { if d70=0 fgr=GREEN }
        { if d70>100 fgr=RED format='! [5]' }
    }
    
    ; Сообщение из txtvals.dat с условным цветом
    { item d38 (height=FONTSIZE_MID)
        GAP_H GAP_V+HEIGHT_MID*2 no yes '<51>'
        { if d38=0 fgr=MAIN_TEXT }
        { if d38>0 fgr=YELLOW }
    }
    
    ; Рамка с условным цветом
    { rect color=MAIN_TEXT
        ifcolors='GREEN if m30=255'
        x=10 y=10 x2=780 y2=580
    }
    
} view

## Правила использования условий

1. Условий может быть несколько в одном элементе
   Они применяются последовательно, последнее истинное условие определяет результат.

2. Порядок условий важен
   Если несколько условий истинны, применяются изменения из всех, но при конфликте побеждает последнее.

3. Параметры не указанные в условии остаются без изменений.

4. Format заменяет весь текст, а не добавляется к нему.

5. В меню условия применяются к конкретному блоку text.

## Типичные применения

| Ситуация | Условие |
|----------|---------|
| Готовность/ошибка | { if m30=255 format='Готово' } { if m30=0 format='Авария' } |
| Активный/неактивный режим | { if d154=1 fgr=GREEN } { if d154=0 fgr=GREY } |
| Диапазон значений | { if d70<0 fgr=BLUE } { if d70>100 fgr=RED } |
| Выбор текста | { if d38=0 format='Норма' } { if d38=1 format='Предупреждение' } { if d38=2 format='Ошибка' } |
| Проверка битов | { if p10&1#0 format='Бит 1 установлен' } |
| Множественные условия | { if ((u195=1)&&(m11=95)&&(m6#0)) format='[6.3]' } |
| Отображение осей | { if IND_KRD0=0 format='' } |
| Символы * и = для осей | { if (m0=6)&&(p10&1#0) format='=' } { if (m0#6)&&(p11&1#0) format='*' } |
