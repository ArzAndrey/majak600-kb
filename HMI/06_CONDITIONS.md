# HMI: Условия (if)

> **⚠️ ВАЖНО ДЛЯ ИИ:**
> Условия в "Маяк-600" пишутся в конце элемента `{ item ... }` и могут изменять атрибуты (цвет, формат и т.д.).
> **НЕ ИСПОЛЬЗУЙТЕ** конструкции `if ... else ... endif` — в "Маяк-600" условия записываются как `{ if ... }` внутри элемента.
> Несколько условий обрабатываются последовательно; при совпадении нескольких условий применяются все соответствующие изменения.
> Все примеры в этом файле — точный синтаксис "Маяк-600".


## Зависимости
- HMI/01_SYNTAX_BASE.md — базовый синтаксис

## Общее описание

Условия позволяют динамически изменять:
- Текст элемента (item)
- Цвет текста и фона
- Параметры отображения (height, width, align)
- Поведение меню

Условия применяются в:
- Элементах item (в конце конструкции)
- Меню (внутри text)
- Графических примитивах (box, rect, line)

---

## Синтаксис условия в item

{ item <источник_данных> (атрибуты)
    <x> <y> <индикация> <нуль-индикация> '<формат>'
    { if <выражение_условия> [атрибут=значение] [format='Текст'] }
    { if <другое_условие> [атрибут=значение] [format='Текст'] }
    ...
}

Условие находится в конце элемента и всегда заключено в фигурные скобки {}.

### Структура
1. `if` — ключевое слово
2. `<выражение_условия>` — логическое выражение
3. `<новые_значения_параметров>` — параметры, применяемые при истинности условия
4. `format='Текст'` — текст, выводимый при истинности условия (необязательно)

## Выражения условия

### Синтаксис
переменная знак_сравнения значение

### Переменные
| Тип | Синтаксис | Пример |
|-----|-----------|--------|
| D-ячейки | `dномер` | `d70` |
| Параметры 0-299 | `uномер` | `u12` |
| Параметры 300-399 | `cномер` | `c124` |
| Параметры 400-499 | `иномер.координата` | `u0.0` |
| Параметры 500-599 | `cномер.координата` | `c15.1` |
| Массив M | `mномер` | `m30` |
| Массив M с битом | `mномер.координата` | `m1.0` |
| Массив I | `iномер` | `i2` |
| Массив I с битом | `iномер.координата` | `i6.1` |
| Массив P | `pномер` | `p0` |
| Переменные HMI | `имя_переменной` | `PRNvar`, `Nvar`, `Xvar` |

### Знаки сравнения
| Знак | Описание |
|------|----------|
| `=` | Равно |
| `#` | Не равно |
| `>` | Больше |
| `<` | Меньше |
| `>=` | Больше или равно |
| `<=` | Меньше или равно |

**Важно:** Оператор `#` используется для "не равно", не используйте `!=` или `<>`.

### Битовые операторы
| Оператор | Описание | Пример |
|----------|----------|--------|
| `&` | Побитовое И | `p10&1#0` (проверка 1-го бита) |

### Примеры выражений
| Выражение | Описание |
|-----------|----------|
| `m30=0` | m30 равно 0 |
| `m30#0` | m30 не равно 0 |
| `m30=255` | m30 равно 255 |
| `d70>100` | d70 больше 100 |
| `u12<50` | параметр 12 меньше 50 |
| `p7=0` | p7 равно 0 |
| `p10&1#0` | проверка 1-го бита p10 |
| `p11&2#0` | проверка 2-го бита p11 |
| `p10&4#0` | проверка 3-го бита p10 (маска 4 = бит 2) |
| `PRNvar=1` | переменная HMI PRNvar равна 1 |
| `PAGEvar#NAN` | переменная HMI PAGEvar не равна NAN |

## Комбинирование условий

### Логические операторы
| Оператор | Описание | Пример |
|----------|----------|--------|
| `&&` | И (AND) | `(u195=1)&&(m11=95)` |
| `\|\|` | ИЛИ (OR) | `(p7=0)\|\|(p30=p2)` |

### Синтаксис
(выражение1) && (выражение2)
(выражение1) || (выражение2)

Скобки используются для группировки условий.

### Примеры
| Условие | Описание |
|---------|----------|
| `(p7=0)&&(p30=p2)` | p7=0 И p30=p2 |
| `(p7=1)\|\|(p30=p8)` | p7=1 ИЛИ p30=p8 |
| `((p7=0)&&(p30=p2))\|\|((p7=1)&&(p30=p8))` | Сложная комбинация |
| `((u195=1)&&(m11=95)&&(m6#0)&&(m6#4)&&(m6#28)&&(u294=0))` | Множественное И |
| `(PRNvar=1)&&(Gvar=1)` | проверка двух переменных HMI |

## Параметры, изменяемые по условию

### В элементе item
| Параметр | Описание |
|----------|----------|
| `height` | Высота шрифта |
| `width` | Ширина строки |
| `fgr` | Цвет текста |
| `bgr` | Цвет фона |
| `invfgr` | Цвет текста при инверсии |
| `invbgr` | Цвет фона при инверсии |
| `align` | Выравнивание |
| `format` | Формат вывода текста (заменяет весь текст) |

### В меню
| Параметр | Описание |
|----------|----------|
| `fgr` | Цвет текста |
| `bgr` | Цвет фона |
| `text` | Текст кнопки (изменяется внутри блока text) |

### В графических примитивах (box, rect, line)
| Параметр | Описание |
|----------|----------|
| `color` | Цвет |
| `ifcolors` | Условное изменение цвета (несколько условий через `;`) |

## Формат 'Текст'

При выполнении условия можно заменить выводимый текст.

### Синтаксис
format='Новый текст'

### Пример
format='ЧПУ: Готово'

## Примеры условий в item

### Пример 1: Простое условие (изменение формата)
{ item # (height=FONTSIZE_SMALL fgr=B3 bgr=B6 align=TEXT_RIGHT)
    BOX_X+BOX_GAP_H+80 BOX_F_Y+BOX_GAP_V no yes 'мм/мин'
    { if m11=95 format=' мм/об' } }

### Пример 2: Несколько условий (переключение между форматами)
{ item # (height=FONTSIZE_SMALL fgr=B3 bgr=B6 align=TEXT_RIGHT)
    BOX_X+BOX_GAP_H+80 BOX_F_Y+BOX_GAP_V no yes 'мм/мин'
    { if m11=95 format=' мм/об' }
    { if m11=941 format=' мкм/с' }
    { if m11=942 format='мм/час' }
    { if m11=943 format=' с/об' } }

### Пример 3: Изменение цвета фона и текста
{ item m30 (invbgr=Error_C invfgr=MAIN_BGR width=106)
    111 GAP_V yes yes 'ЧПУ: Авария'
    { if m30=255 invbgr=GREEN format='ЧПУ: Готово' } }

### Пример 4: Изменение цвета текста
{ item m20 (fgr=B3 bgr=B7 width=160)
    111 GAP_V no yes '            '
    { if m20#0 format='<24>' } 
    { if m20=0 fgr=MAIN_BGR } }

### Пример 5: Скрытие элемента (перекрытие)
{ item # (width=1 bgr=MAIN_BGR)
    560 16 no yes ' '
    { if m30=0 bgr=MAIN_BGR format=' '}
    { if m30=255 bgr=MAIN_BGR format=' '} }

### Пример 6: Сложное условие с логическими операторами
{ item # (fgr=B3 bgr=B6)
    BOX_X+BOX_GAP_H+28 BOX_F_Y+BOX_GAP_V+HEIGHT_MID+0.5 no yes '&m17[5]'
    { if ((u195=1)&&(m11=95)&&(m6#0)&&(m6#4)&&(m6#28)&&(u294=0))
        format='&(m17*1000)/d77[6.3]' }
    { if ((u195=1)&&(m11=95)&&(m6#0)&&(m6#4)&&(m6#28)&&(u294=1))
        format='&(m17*1000)/i34[6.3]' } }

### Пример 7: Условие с битовой операцией
{ item # (height=20) 100 50 no yes ''
    { if (m0=6)&&(IND_KRD0=1)&&(p10&1#0) format='=' }
    { if (m0#6)&&(IND_KRD0=1)&&(p11&1#0) format='*' } }

### Пример 8: Условие с диапазоном значений
{ item d70 (height=20 fgr=MAIN_TEXT bgr=MAIN_BGR)
    100 50 no yes '[5]'
    { if d70>100 fgr=RED bgr=YELLOW }
    { if d70<0 fgr=BLUE format='Ошибка' } }

### Пример 9: Множественное условие с комбинацией
{ item d38 (height=20)
    100 50 no yes 'Статус: [3]'
    { if (d38=0)||(d38=255) fgr=GREEN }
    { if (d38>0)&&(d38<255) fgr=RED } }

### Пример 10: Условие с проверкой D-ячейки
{ item d77 (fgr=B3 bgr=B6)
    BOX_X+BOX_GAP_H+28 BOX_S_Y+BOX_GAP_V+HEIGHT_MID-1 no yes '[5]'
    { if d77=0 fgr=MAIN_TEXT }
    { if d77>1000 fgr=RED } }

### Пример 11: Условие с переменной HMI (проверка наличия значения)
{ item Nvar (width=70) X Y no yes 'N[5]'
    { if Nvar=NAN format='N ---' } }

## Правила использования условий

1. **Несколько условий в одном элементе**
   Условия применяются последовательно. Если несколько условий истинны, применяются изменения из всех. При конфликте (например, два условия изменяют один атрибут) побеждает последнее.

2. **Порядок условий важен**
   Рекомендуется располагать условия от более общих к более специфичным.

3. **Параметры, не указанные в условии, остаются без изменений.**

4. **Format заменяет весь текст, а не добавляется к нему.**

5. **В меню условия применяются к конкретному блоку text.**
   Для изменения текста кнопки используется синтаксис `{ text "Текст" { if условие text="Новый текст" } }`.

6. **Условия в меню могут изменять цвет и текст одновременно.**

## Типичные применения

| Ситуация | Условие |
|----------|---------|
| Готовность/ошибка | `{ if m30=255 format='Готово' }` `{ if m30=0 format='Авария' }` |
| Активный/неактивный режим | `{ if d154=1 fgr=GREEN }` `{ if d154=0 fgr=GREY }` |
| Диапазон значений | `{ if d70<0 fgr=BLUE }` `{ if d70>100 fgr=RED }` |
| Выбор текста (несколько вариантов) | `{ if d38=0 format='Норма' }` `{ if d38=1 format='Предупреждение' }` `{ if d38=2 format='Ошибка' }` |
| Проверка битов | `{ if p10&1#0 format='Бит 1 установлен' }` |
| Множественные условия | `{ if ((u195=1)&&(m11=95)&&(m6#0)) format='[6.3]' }` |
| Отображение осей (скрытие) | `{ if IND_KRD0=0 format='' }` |
| Символы * и = для осей | `{ if (m0=6)&&(p10&1#0) format='=' }` `{ if (m0#6)&&(p11&1#0) format='*' }` |
| Переключение единиц измерения | `{ if m11=95 format=' мм/об' }` `{ if m11=941 format=' мкм/с' }` |
| **Изменение текста кнопки меню** | `{ text "Записать$в УП" { if PRNvar=1 text="Записать$кадр" } }` |
| **Скрытие/показ подсказки меню** | `{ f4 { text "Пред.$G-функция" { if PAGEvar#NAN fgr=REG_F } } }` |
| **Проверка наличия значения** | `{ if Nvar=NAN format='N ---' }` |
| **Комбинирование условий в меню** | `{ text " &0 " { if d406=1 fgr=WARNING } { if IND_KRD0=0 fgr=BG_MAIN } }` |

## Особенности и рекомендации

1. **Используйте скобки для сложных условий** — это улучшает читаемость и гарантирует правильный порядок операций.

2. **Оператор `#` означает "не равно"** — не используйте `!=` или `<>`.

3. **Для проверки битов используйте `&`** — например, `p10&4#0` проверяет, установлен ли третий бит (маска 4).

4. **Условия могут изменять только атрибуты текущего элемента** — нельзя изменить другой элемент или переменную.

5. **При переключении формата указывайте полный текст** — условие заменяет весь format, а не добавляет к нему.

6. **Для скрытия элемента используйте `format=' '` или `width=1`** — элемент остаётся в разметке, но не отображается.

7. **Несколько условий обрабатываются последовательно** — если нужно, чтобы применялось только одно, используйте взаимоисключающие условия.

8. **В меню для изменения текста кнопки используйте `text=` внутри `{ if ... }`** — например, `{ text "Текст" { if условие text="Новый текст" } }`.

9. **Для изменения цвета в меню используйте `fgr=` и `bgr=` внутри `{ if ... }`** — например, `{ text "Текст" { if условие fgr=RED } }`.

10. **В меню можно комбинировать несколько условий в одном блоке text** — например, `{ text " &0 " { if d406=1 fgr=WARNING } { if IND_KRD0=0 fgr=BG_MAIN } }`.

## Типичные ошибки

| Неправильно | Правильно | Пояснение |
|-------------|-----------|-----------|
| `{ if m30!=255 }` | `{ if m30#255 }` | Используйте `#` вместо `!=` |
| `{ if (u195=1 and m11=95) }` | `{ if (u195=1)&&(m11=95) }` | Используйте `&&` вместо `and` |
| `{ if m6=0 }` внутри format | `{ if m6=0 format='...' }` | Формат изменяется через атрибут format |
| `{ if ... }` вне item | `{ item ... { if ... } }` | Условия только внутри элемента |
| `{ if m30=255 }` без format | `{ if m30=255 format='Готово' }` | Условие без изменений не имеет смысла |
| `{ if (p10&1) }` | `{ if p10&1#0 }` | Для проверки бита обязательно `#0` |
| `{ text "Текст" } { if PRNvar=1 text="Новый" }` | `{ text "Текст" { if PRNvar=1 text="Новый" } }` | Условие должно быть внутри блока text, а не после него |
| `{ text "Текст" { if PRNvar=1 } }` | `{ text "Текст" { if PRNvar=1 text="Новый" } }` | В условии для text нужно указать text= |
| `{ if PAGEvar=NAN fgr=REG_F }` в меню без text | `{ text "Текст" { if PAGEvar=NAN fgr=REG_F } }` | В меню условия применяются к блоку text |
