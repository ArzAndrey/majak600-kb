# База знаний УЧПУ "Маяк-600"

### ⚠️ ВАЖНО ДЛЯ ИИ
При генерации кода **строго соблюдай синтаксис** УЧПУ "Маяк-600":
- Все конструкции в `{ }`
- Закрывающие конструкции повторяют открывающие
- Используй константы из `CONFIG/03_COLORS.md` и `CONFIG/04_CONSTANTS.md`
- Не используй синтаксис других ЧПУ (Fanuc, Siemens и т.д.)

## Описание
Данная база знаний содержит полную документацию по языкам программирования УЧПУ "Маяк-600":
- **HMI (язык разметки интерфейса)** — описание экранных форм, элементов, меню
- **ПЭС (PLC язык)** — программа электроавтоматики станка

База структурирована для использования ИИ (включая DeepSeek, ChatGPT, Claude и др.) для генерации кода по описанию или изображению экрана.

## Использование

### Для ИИ (в новом чате)
1. Скопируйте этот индексный файл в контекст
2. При необходимости запросите конкретные модули по их названиям
3. Используйте `QUICK_REFERENCE.md` для быстрой генерации

### Для человека
Используйте как справочник при разработке интерфейсов и ПЭС для УЧПУ "Маяк-600"

## Режимы работы (номера форм)

| Номер form | Режим |
|------------|-------|
| 0 | Автомат |
| 1 | Ручной |
| 2 | Выход в ноль |
| 6 | Выход в точку |
| 7 | Преднабор |
| 8 | Фиксированные перемещения |
