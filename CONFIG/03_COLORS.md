## Общее описание

Файл colors.dat содержит определения цветовых констант, используемых в HMI (экранных формах, элементах, меню, графике). Цвета задаются в формате RGB(R,G,B) и могут использоваться во всех конструкциях языка разметки интерфейса.

## Использование в HMI

### В элементе item
{ item # (height=FONTSIZE_MID fgr=AN_C bgr=MAIN_BGR)
    GAP_H GAP_V yes yes 'Автомат'
}

### В элементе item с условием
{ item m30 (invbgr=Error_C invfgr=MAIN_BGR width=106)
    111 GAP_V yes yes 'ЧПУ: Авария'
    { if m30=255 invbgr=GREEN format='ЧПУ: Готово' }
}

### В блоке box
{ box color=REG_F x=BOX_X y=BOX_F_Y x2=BOX_W y2=BOX_F_H }

### В блоке rect
{ rect color=MAIN_TEXT x=10 y=10 x2=780 y2=580 }

### В меню
{ text "Пуск" (fgr=MENU_FGR bgr=MENU_BGR) }

## Рекомендации

1. **Используйте системные константы** (MAIN_BGR, Error_C и т.д.) вместо прямых RGB для единообразия

2. **При создании новых экранов** придерживайтесь существующей цветовой схемы

3. **Для выделения активных элементов** используйте CPARAM (жёлтый)

4. **Для неактивных элементов** используйте CPARAM3 (серо-голубой)

5. **Цвета B1-B10** образуют градиентную шкалу от светлого к тёмному

6. **При изменении цветов** учитывайте контрастность для читаемости текста
