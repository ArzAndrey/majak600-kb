## Зависимости
- HMI/01_SYNTAX_BASE.md — базовый синтаксис HMI
- HMI/02_STRUCTURE.md — структура Form, Grid, View
- HMI/03_ELEMENTS.md — элементы item, box, rect

---

## Общее описание

Файл defaults.dat содержит числовые константы, используемые в HMI для:
- Отступов и позиционирования элементов
- Размеров шрифтов
- Параметров сетки (grid)
- Размеров блоков (F, S, T, GM, маховик)
- Координат вывода элементов

Константы могут использоваться во всех конструкциях языка разметки интерфейса.

## Блок default в defaults.dat

Файл defaults.dat также содержит блок default, задающий параметры по умолчанию для всех элементов.

{ default
    bgr = RGB(0,0,0)
    fgr = RGB(255,255,255)
    invbgr = RGB(255,255,255)
    invfgr = RGB(0,0,0)
    selbgr = RGB(255,255,255)
    selfgr = RGB(0,0,0)
    align = TEXT_LEFT
    height = 5.0
    xmax = 100
    ymax = 100

    title_h = 7
    frame_h = 82

    f13 = 41013h ; переход вверх по вертикальному меню
    f14 = 41015h ; продолжение вниз по вертикальному меню
    f15 = 41012h ; возврат по горизонтальному меню
    f16 = 41014h ; продолжение по горизонтальному меню

    menu = 0
    msgfgr = RGB(255,255,255)
    msgbgr = 0
    errfgr = RGB(0,255,0)
    errbgr = 0
    border = RGB(0,255,0)
    fonts = 'IBMPlexSans'
} default

## Использование констант в HMI

### Пример 1: Отступы
{ item # () GAP_H GAP_V yes yes 'Автомат' }

### Пример 2: Размер шрифта
{ item # (height=FONTSIZE_MID) GAP_H GAP_V yes yes 'Автомат' }

### Пример 3: Блок F
{ box color=REG_F x=BOX_X y=BOX_F_Y x2=BOX_W y2=BOX_F_H }
{ item # (height=FONTSIZE_MID) BOX_X+BOX_GAP_H BOX_F_Y+BOX_GAP_V no yes 'F' }

### Пример 4: Блок координат
{ item m1.0 (height=FONTSIZE_MID)
    D_COORD_1+COORD_W*0 BLOCK_COORD_Y no yes '[9.3]' }

### Пример 5: Сетка
{ grid g_main (grid_rows=DEF_GRID_ROWS grid_cols=DEF_GRID_COLS)
    { view (grid_x=DEF_GRID_X grid_y=DEF_GRID_Y 
            grid_w=DEF_GRID_W grid_h=DEF_GRID_H)
        ...
    } view
} grid

### Пример 6: Константы в атрибутах формы

Константы можно использовать в атрибутах `xmax` и `ymax` формы или вида:

{ form 1 (xmax=FORM1_Xmax, ymax=FORM1_Ymax)
    { view () ... } view
} form

Где `FORM1_Xmax` и `FORM1_Ymax` — константы, определенные в defaults.dat или в комментариях файла.

**Примечание:** Константы могут быть определены как в файле defaults.dat, так и в самом HMI-файле через директиву `#ИМЯ=значение`.
---

## Рекомендации

1. **Используйте константы вместо "магических чисел"** для обеспечения единообразия

2. **При создании новых экранов** придерживайтесь существующих отступов (GAP_H, GAP_V)

3. **Для выравнивания блоков** используйте BOX_X, BOX_W и производные координаты

4. **Для позиционирования элементов внутри блока** используйте BOX_GAP_H, BOX_GAP_V

5. **Размеры шрифтов** выбирайте из FONTSIZE_SMALL, FONTSIZE_MID, FONTSIZE_BIG, FONTSIZE_BIGGER

6. **При создании пользовательских сеток** используйте DEF_GRID_* константы как основу
