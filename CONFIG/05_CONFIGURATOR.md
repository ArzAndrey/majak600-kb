## Общее описание

Конфигуратор оборудования УЧПУ "Маяк" предназначен для настройки периферийных устройств и сетевого взаимодействия. Он определяет, какие физические устройства подключены к системе, их параметры и привязку к логическим адресам.

Вход в конфигуратор осуществляется через консольный интерфейс сочетанием клавиш Ctrl+N.

## Раздел "Оборудование"

### Подготовка к конфигурации
Перед настройкой необходимо определить:
1. Тип системы ЧПУ
2. Состав устройств
3. Скорость каналов CAN, RS-485, SSI
4. Адреса CAN-устройств (Node Id 1-127)
5. Порядок подключения устройств на EtherCAT
6. Номера платомест
7. Используемые ресурсы устройств и их адреса
8. Привязку управления и обратной связи по координатам

### Последовательность конфигурации
1. Выбор главного устройства
2. Добавление периферийных устройств
3. Конфигурация параметров устройств
4. Конфигурация таблиц ввода-вывода

## Подключение устройств

### Платоместа
Устройства устанавливаются в физические слоты. Адрес задаётся автоматически.

### EtherCAT
Адрес определяется положением устройства в цепи (0, 1, 2...). Положение адресных переключателей не учитывается.

### CAN
Адрес устройства (Node Id) от 1 до 127. Настраивается в конфигураторе и на устройстве.

### RS-485
Адрес устройства от 1 до 127.

### SSI
Номер канала ДОС (1-5) для модуля Маяк-ВВ 405A.

## Конфигурация устройства

### Поля настройки
| Поле | Описание |
|------|----------|
| Имя устройства | Символьное имя (до 32 знаков), используется в ПЭС и индикации ошибок |
| Включен | Признак учёта устройства контроллером периферии |
| Конфигурация | Параметры устройства (индивидуальны для каждого типа) |
| Данные | Включение/отключение ресурсов устройства |
| Адрес | Адрес устройства в сети |

### Параметры данных (Делитель/Смещение)
| Значение | Описание |
|----------|----------|
| Делитель=1, Смещение=0 | Данные передаются/опрашиваются каждое прерывание |
| Делитель>1 | Данные передаются/опрашиваются с указанной периодичностью |
| Делитель=0 (вход CAN) | Контроль поступления пакета каждое прерывание |

## Настройка регулятора приводов

### Последовательность настройки
1. Установить R508.K = 10000 (10 мм допуска)
2. Установить R509.K = 10, R510.K = 0, R511.K = 0
3. Установить R523.K биты 1 и 3 (&10)
4. Включить привод
5. Подобрать R509.K (КП) для рассогласования ~200 мкм при 240 мм/мин
6. Подобрать R510.K (КСО) для минимизации рассогласования
7. Подобрать R511.K (КОУ) для минимизации рассогласования при разгоне/торможении

## Особые случаи настройки

### Блок МИ-064 или ЦАП-5И
- Период прерываний блока INT PERIOD (6000 мкс)
- QUERYPERIOD = 31000 (3.1 мс)
- КП подстраивает свои прерывания под блок

### Блок Маяк-ВВ 404С
- Всегда задействовать ДОС3
- Отключать неиспользуемые ДОС
- CANxSYNCDIVIDER = 1

### Электронные штурвалы (маховики)
- Отключать неиспользуемые ресурсы "Энкодер" или "Маховик"
- Иначе они участвуют в нумерации маховиков

## Диагностика после настройки

### Таблица QSEL (g0-g41)
| Ячейка | Назначение | Норма |
|--------|------------|-------|
| g(0) | Период прерываний | Соответствует QUERYPERIOD |
| g(1) | Время измерения | < ½ g(0) с запасом |
| g(2) | Время контура | < ¾ g(0) |
| g(3) | Уровень шума RTE | 0 (наилучшее) |
| g(5), g(7) | Размер пакетов RTE | Меняющиеся значения |
| g(9), g(10) | Ошибки CRC RTE | 0 |
| g(15)-g(34) | CAN/EtherCAT/RS485 RX/TX | Меняющиеся значения при наличии устройств |

## Рекомендации

1. **Перед настройкой** подготовьте полный список устройств с адресами и ресурсами

2. **Отключайте неиспользуемые устройства** в конфигураторе, чтобы избежать ошибок связи

3. **Для каждого устройства** обязательно зайти в его конфигурацию и выйти с сохранением

4. **После изменения конфигурации** выполните перезагрузку УЧПУ

5. **Имена устройств** должны быть информативными и краткими (используются в ПЭС)

6. **Проверяйте время измерения (g1)** и время контура (g2) после настройки

7. **При настройке регулятора** увеличивайте КП постепенно, избегая возбуждения привода
