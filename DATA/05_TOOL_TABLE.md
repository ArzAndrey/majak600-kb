## Общее описание

Таблица инструментов (ТИ) хранит данные о каждом инструменте, используемом в станке: длины, радиусы, коррекции, ориентацию и другие параметры. Таблица состоит из 200 элементов (T0-T199), каждый элемент содержит 10 значений (индексы 0-9).

---

## Структура таблицы инструментов

| Элемент | Назначение |
|---------|------------|
| T0 - T199 | Данные по инструментам |

### Индексы элемента
| Индекс | Фрезерный вариант | Токарный вариант |
|--------|-------------------|------------------|
| 0 | Коррекция по X (DZ) | Коррекция по X (DX) |
| 1 | Коррекция по Y | Коррекция по Z (DZ) |
| 2 | Коррекция по Z | Длина по X (X) |
| 3 | Длина по X | Длина по Z (Z) |
| 4 | Длина по Y | Ориентация (P) |
| 5 | Длина по Z | Радиус (R) |
| 6 | Радиус (R) | Коррекция радиуса (DR) |
| 7 | Ориентация | (резерв) |
| 8 | (резерв) | (резерв) |
| 9 | (резерв) | (резерв) |

---

## Настройка таблицы инструментов

В файле machine.cfg настраиваются параметры таблицы инструментов:

### Пример для фрезерного станка
%tools
xdim=16              ; количество элементов ТИ (0-15)
ydim=3               ; количество значений в элементе
DZ=Корр.по Z         ; комментарий к индексу 0
Z=Длина по Z         ; комментарий к индексу 1
R=Радиус             ; комментарий к индексу 2

### Пример для токарного станка
%tools
xdim=5               ; количество элементов ТИ (0-4)
ydim=7               ; количество значений в элементе
DX=Корр.по X         ; комментарий к индексу 0
DZ=Корр.по Z         ; комментарий к индексу 1
X=Длина по X         ; комментарий к индексу 2
Z=Длина по Z         ; комментарий к индексу 3
P=Ориентация         ; комментарий к индексу 4
R=Радиус             ; комментарий к индексу 5
DR=Корр.радиус       ; комментарий к индексу 6

## Использование в ПЭС

### Синтаксис
Tномер.индекс
T[номер][индекс]

### Чтение значения
length_x = T5.2       ; длина инструмента 5 по X
radius = T5.6         ; радиус инструмента 5

### Запись значения
T5.2 = 100000         ; установить длину 100 мм
T5.6 = 5000           ; установить радиус 5 мм

### Косвенная адресация
my_tool = p8          ; текущий номер инструмента
my_length = T[my_tool][2]

---

## Параметры, связанные с таблицей инструментов

| Параметр | Назначение |
|----------|------------|
| R198 | Работа с таблицей инструментов (0-нет, 1-есть, 2-только ТИ) |
| R8 | Текущий номер инструмента при работе с ТИ |
| P(8) | Текущий номер инструмента |
| P(20) | Индекс таблицы инструментов для доступа |
| P(21)-P(29) | Значения с 1-го по 9-й элемента ТИ (по индексу из P20) |

## Работа с таблицей инструментов из ПЭС

### Чтение длины инструмента по X
:1
    my_tool = m5         ; текущий инструмент
    my_length = T[my_tool][2]   ; длина по X
    if my_length = 0
        warn "Не задана длина инструмента"
    fi

### Чтение радиуса инструмента
:1
    my_tool = m5
    my_radius = T[my_tool][6]    ; радиус
    if my_radius = 0
        warn "Не задан радиус инструмента"
    fi

### Запись данных инструмента после измерения
:88
    my_tool = p8
    T[my_tool][2] = d78           ; длина по X
    T[my_tool][6] = d79           ; радиус
    tsave()                       ; сохранить таблицу
    kill

### Использование P20-P29 для доступа
:10
    p20 = p8                      ; установить индекс
    my_length = p21               ; длина (первое значение)
    my_radius = p22               ; радиус (второе значение)

## Системные функции для работы с ТИ

### tsave()
Сохраняет таблицу инструментов. Необходимо выполнять однократно после внесения изменений.

Пример:
T5.2 = 100000
tsave()

### rsave()
Сохраняет параметры. Необходимо выполнять однократно после внесения изменений в ТИ.

Пример:
T5.2 = 100000
tsave()
rsave()

## Примеры использования

### Пример 1: Смена инструмента с проверкой
:106                     ; M6 - смена инструмента
    my_new_tool = p2     ; новый инструмент из УП
    my_old_tool = p8     ; текущий инструмент
    
    if my_new_tool = my_old_tool
        ; инструмент уже установлен
        kill
    fi
    
    ; проверка наличия данных в ТИ
    if T[my_new_tool][2] = 0
        error "Не задана длина инструмента", my_new_tool
        kill
    fi
    
    ; смена инструмента
    A3.3 = 1             ; разжим патрона
    wait(500)
    
    ; команда на смену
    d15 = my_new_tool    ; установить новый номер
    
    wait(2000)
    
    A3.4 = 1             ; зажим патрона
    wait(500)
    
    ; сохранение
    p8 = my_new_tool
    kill

### Пример 2: Измерение инструмента
:120
    ; измерение длины инструмента
    my_tool = p8
    
    ; подвод к датчику
    d78 = 100000         ; 100 мм
    d40 = 100            ; 100 мм/мин
    d47.0 = 1            ; направление +
    
    while d40 != 0
        wait
    wend
    
    ; запись измеренного значения
    T[my_tool][2] = m1.0
    tsave()
    
    msg "Инструмент [3] измерен, длина: [9.3] мм", my_tool, m1.0
    kill

### Пример 3: Вывод данных инструмента на экран
:1
    my_tool = m5
    if my_tool != 0
        my_length = T[my_tool][2] / 1000
        my_radius = T[my_tool][6] / 1000
        ; данные доступны для HMI
    fi
    wait(100)
    goto 1

### Пример 4: Коррекция радиуса инструмента
:130
    my_tool = p8
    my_offset = d70      ; коррекция из ПЭС
    
    T[my_tool][6] = T[my_tool][6] + my_offset
    tsave()
    
    msg "Коррекция радиуса инструмента [3]: [6.3] мм", my_tool, T[my_tool][6]
    kill

### Пример 5: Проверка наличия всех инструментов
:1
    my_ok = 1
    my_i = 0
    
    while my_i < 16
        if T[my_i][2] = 0
            my_ok = 0
            warn "Не задана длина инструмента [2]", my_i
        fi
        my_i = my_i + 1
    wend
    
    if my_ok = 1
        msg "Все инструменты настроены"
    fi
    
    wait(10000)
    goto 1

## Работа с таблицей инструментов в HMI

Для отображения данных ТИ на экране используются элементы item с переменными Tномер.индекс.

### Пример вывода данных инструмента
{ item T5.2 (height=FONTSIZE_MID width=100)
    BOX_X+BOX_GAP_H BOX_F_Y+BOX_GAP_V no yes '[9.3]'
}

### Пример вывода радиуса с текстом
{ item # (height=FONTSIZE_MID)
    BOX_X+BOX_GAP_H BOX_F_Y+BOX_GAP_V+HEIGHT_MID no yes
    'Радиус: [9.3] мм' T5.6
}

## Рекомендации

1. **Всегда выполняйте tsave()** после изменения таблицы инструментов, иначе изменения не сохранятся при перезагрузке

2. **Используйте P20-P29** для быстрого доступа к значениям ТИ по индексу

3. **Проверяйте наличие данных** в ТИ перед использованием инструмента

4. **Для фрезерных станков** используются коррекции по длине (DZ) и радиус (R)

5. **Для токарных станков** используются коррекции по X и Z, длины по X и Z, радиус и коррекция радиуса

6. **Номер инструмента в УП (Txx)** записывается в P2, а после отработки M6 — в P8

7. **При работе без таблицы инструментов** (R198=0) коррекции задаются в УП непосредственно
