## Общее описание

D-ячейки (D0-D999) — это универсальные ячейки памяти для обмена данными между БПрО и ПЭС. Они могут использоваться для:
- Хранения промежуточных данных
- Упрощённого доступа к другим данным (через файл hash_d.cfg)
- Управления режимами работы УЧПУ

---

## Структура D-ячеек

### Диапазоны
| Диапазон | Назначение |
|----------|------------|
| D0-D104 | Обмен с БПрО и системной ПЭС |
| D115-D299 | Внутреннее использование ПЭС |
| D300-D449 | Системная ПЭС и расширенные функции |
| D2000-D2599 | Обращение к параметрам 0-599 |

## Использование D-ячеек в ПЭС

### Запись значения
d10 = 1
d40 = 100
d78 = 5000
d49 = 1

### Чтение значения
my_speed = d40
if d10 = 1
    ; УП запущена
fi

### Работа с битами
d23.0 = 1           ; установить бит 0
d23.1 = 0           ; сбросить бит 1
if d23.0 = 1        ; проверить бит

### Маскированная запись
d23 & 1 = 1         ; установить бит 0
d23 & 2 = 0         ; сбросить бит 1

### Присвоение группе
D1, 10 = 0          ; очистить D1-D10
D1.4, 10 = 1        ; установить бит 4 в D1-D10

## Привязка D-ячеек к массивам БПрО

В файле hash_d.cfg определяется соответствие D-ячеек элементам массивов БПрО.

Пример из hash_d.cfg:
D0 i(33) ; Код на ЦАП
D1 p(1) ; Память технологической команды S
D2 p(2) ; Память технологической команды T
D10 i(3) ; Отработка УП 1-отработка,0-стоп
D38 m(28) ; Код сообщения
D39 m(29) ; Код ошибки
D40 i(1.0) ; Скорость по X при перемещениях от ПЭС
D78 i(3.0) ; Величина перемещения по X
D350 m(7.0) ; Расстояние до маркера по X

## Типичные сценарии использования

### Запуск УП
d10 = 1

### Останов УП
d10 = 0

### Включение шпинделя
d49 = 1

### Выключение шпинделя
d49 = 0

### Смена режима работы
d11 = 0    ; Автомат
d11 = 1    ; Ручной
d11 = 2    ; Выход в ноль

### Фиксированное перемещение по X
d78 = 10000    ; перемещение 10 мм
d40 = 500      ; скорость 500 мм/мин
d47.0 = 1      ; направление "+X"
; ожидание окончания
while d40 != 0
    wait
wend

### Вывод сообщения об ошибке
d39 = 251      ; код ошибки
error "Ошибка привода"

### Проверка готовности УЧПУ
if m30 = 255
    ; УЧПУ готово
fi
