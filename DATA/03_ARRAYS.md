## Общее описание

Массивы I(), M(), P() — это внутренние данные БПрО, используемые для обмена информацией между БПрО и ПЭС. Они содержат данные интерполяции, индикации, состояния станка и другую служебную информацию.

## Использование массивов в ПЭС

### Чтение одномерного массива
mode = m0                 ; текущий режим
prog_num = m1             ; номер УП
frame_num = m2            ; номер кадра
tool_num = m5             ; номер инструмента
speed = m17               ; текущая скорость
error_code = m29          ; код ошибки

### Чтение двумерного массива
err_x = m0.0              ; рассогласование по X
err_y = m0.1              ; рассогласование по Y
current_x = m1.0          ; текущее положение X
real_x = m14.0            ; реальное положение X

### Запись в массивы
p6 = 1                    ; смена режима на "Ручной"
p3 = 1                    ; сброс УЧПУ
p13.0 = 1                 ; выход в 0 по X в "+"

## Примеры использования

### Пример 1: Определение текущего режима
if m0 = 0
    ; режим "Автомат"
elif m0 = 1
    ; режим "Ручной"
fi

### Пример 2: Проверка готовности УЧПУ
if m30 = 255
    ; УЧПУ готово
else
    ; УЧПУ не готово
fi

### Пример 3: Чтение текущего положения
my_x = m1.0
my_y = m1.1
my_z = m1.2

### Пример 4: Ожидание окончания перемещения
while m2.0 != 0
    wait
wend

### Пример 5: Проверка рассогласования
if abs(m0.0) > R508.0
    error "Превышено рассогласование по X"
fi

### Пример 6: Смена режима из ПЭС
p6 = 1    ; переключение в "Ручной"

### Пример 7: Сброс УЧПУ при аварии
if A19.0 = 0
    p3 = 1
fi

### Пример 8: Чтение номера кадра
frame = m2
msg "Кадр: [5]" frame

### Пример 9: Чтение номера инструмента
tool = m5
msg "Инструмент: [3]" tool
