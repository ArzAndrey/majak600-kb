## Общее описание

Параметры R0-R599 хранят настройки станка, коррекции, смещения систем координат и другие данные. Параметры делятся на одномерные (R0-R399) и двумерные (R400-R599) с привязкой к координатам.

---

## Структура параметров

### Одномерные параметры (R0-R399)
| Диапазон | Назначение |
|----------|------------|
| R0-R184 | Параметры, используемые для УП |
| R185-R299 | Параметры настройки УЧПУ |
| R300-R399 | Станочные параметры (электроавтоматика) |

### Двумерные параметры (R400-R599)
Каждому номеру параметра соответствует несколько ячеек по числу координат. Нумерация координат начинается с 0 (X).

| Диапазон | Назначение |
|----------|------------|
| R400-R409 | Смещение систем координат (G54-G59) |
| R410-R419 | Зоны исключения |
| R430-R431 | Координаты касания (G31) |
| R440-R489 | Смещение СК (G540-G589) |
| R490-R492 | Параметры постоянных циклов |
| R500-R599 | Геометрические и физические параметры станка |

## Битовая структура параметров

### R523.K (Вид работающих координат)
| Бит | Маска | Назначение |
|-----|-------|------------|
| 0 | 1 | Включен датчик обратной связи |
| 1 | 2 | Имеется выход на привод |
| 2 | 4 | Шпиндель |
| 3 | 8 | Отключение КП (только для отладки) |
| 4 | 16 | Датчик обратной связи абсолютный |
| 5 | 32 | Управление приводом по положению |
| 6 | 64 | Управление координатой по двум датчикам |

### R522.K (Параметры выхода в ноль)
| Бит | Маска | Назначение |
|-----|-------|------------|
| 0 | 1 | Смена направления съезда с КВ зоны останова |
| 1 | 2 | Быстрый выход в 0 для индуктивных датчиков |
| 2 | 4 | Выход в 0 для шаговых с фазой 0 |
| 3 | 8 | Смена направления выхода в 0 с пульта ЧПУ |

### R194 (Отмена MST-функций)
| Бит | Маска | Назначение |
|-----|-------|------------|
| 0 | 1 | Отмена M-функций |
| 1 | 2 | Отмена S-функций |
| 2 | 4 | Отмена T-функций |

## Использование параметров в ПЭС

### Чтение одномерного параметра
max_speed = R503.0      ; максимальная скорость по X
accel = R505.0          ; ускорение по X
zone = R506.0           ; зона допуска по X

### Чтение двумерного параметра
offset_x = R400.0       ; смещение G54 по X
offset_y = R400.1       ; смещение G54 по Y
offset_z = R400.2       ; смещение G54 по Z

### Запись параметра
R503.0 = 5000           ; установить максимальную скорость 5000 мм/мин
R506.0 = 10             ; установить зону допуска 10 мкм

### Косвенная адресация
my_axis = 0
my_speed = R503[my_axis]    ; скорость по текущей оси

### Пример 1: Настройка параметров координаты X

R505.0 = 500        ; ускорение 500 (мм/мин)/с
R508.0 = 50         ; допустимое рассогласование 50 мкм
R509.0 = 100        ; КП
R510.0 = 50         ; КСО
R523.0 = 3          ; биты 0 и 1 установлены (ДОС и привод)

R400.0 = 10000      ; смещение X 10 мм
R400.1 = 5000       ; смещение Y 5 мм
