# Примеры файлов HMI для УЧПУ "Маяк-600"

## Описание

В этой папке собраны реальные примеры файлов HMI из проекта, демонстрирующие различные конструкции языка. Файлы могут использоваться как справочный материал и для тестирования.

---

## Содержание

### Базовые элементы (box_*.dat)
Файлы, описывающие отдельные визуальные блоки (зоны) интерфейса.

| Файл | Назначение |
|------|------------|
| box_f_avt.dat | Зона F для автоматических режимов (подача, %F) |
| box_f_nal.dat | Зона F для ручных режимов |
| box_s_avt.dat | Зона S для автоматических режимов (шпиндель, %S, SA) |
| box_s_nal.dat | Зона S для ручных режимов |
| box_r.dat | Зона БХ (быстрая подача) |
| box_t.dat | Зона T (инструмент, привязка) |
| box_gm.dat | Зона G и M (G-коды, M-функции) |
| box_mhv.dat | Зона маховика (ручное управление) |
| box_up.dat | Блок управляющей программы (текст УП) |
| box_prn.dat | Блок УП для режима Преднабор |
| box_fix.dat | Величина фиксированного перемещения |

### Таблицы координат (box_*_coord.dat)
Файлы с отображением координат осей.

| Файл | Назначение |
|------|------------|
| box_name_s.dat | Названия осей с символами * и = |
| box_ost.dat | Остаток пути по осям |
| box_rsg.dat | Рассогласование по осям |
| box_ost_fix.dat | Остаток пути для режима Фикс.перем. |
| box_ost_s.dat | Остаток пути (упрощенный) |
| box_rsg_s.dat | Рассогласование (упрощенный) |

### Комплексные блоки (box_*_*.dat)
Файлы, объединяющие несколько зон (Заготовка, Остаток пути, Станок, Рассогласование, Нагрузка).

| Файл | Назначение |
|------|------------|
| box_ZOSR_avt.dat | Заготовка + Остаток пути + Станок + Рассогласование (Автомат) |
| box_ZOSR_fix.dat | Заготовка + Остаток пути + Станок + Рассогласование (Фикс.перем.) |
| box_ZO_avt.dat | Заготовка + Остаток пути (Автомат) |
| box_ZO_fix.dat | Заготовка + Остаток пути (Фикс.перем.) |
| box_SR_avt.dat | Станок + Рассогласование (Автомат) |
| box_SR_fix.dat | Станок + Рассогласование (Фикс.перем.) |
| box_SR_nal.dat | Станок + Рассогласование (Ручной) |
| box_SR_vi.dat | Станок + Рассогласование (Выход в ноль/точку) |
| box_ZL_avt.dat | Заготовка + Нагрузка (Автомат) |
| box_ZL_fix.dat | Заготовка + Нагрузка (Фикс.перем.) |
| box_ZL_nal.dat | Заготовка + Нагрузка (Ручной) |
| box_ZL_vi.dat | Заготовка + Нагрузка (Выход в ноль/точку) |
| box_ZR_nal.dat | Заготовка + Рассогласование (Ручной) |
| box_ZR_vi.dat | Заготовка + Рассогласование (Выход в ноль/точку) |
| box_ZS_nal.dat | Заготовка + Станок (Ручной) |
| box_ZS_prn.dat | Заготовка + Станок (Преднабор) |
| box_ZSR_nal.dat | Заготовка + Станок + Рассогласование (Ручной) |
| box_ZSR_vi.dat | Заготовка + Станок + Рассогласование (Выход в ноль/точку) |

### Формы (form*.dat)
Основные экраны интерфейса.

| Файл | Назначение |
|------|------------|
| form0.dat | Индикация режима "Автомат" |
| form1.dat | Индикация режима "Ручной" |
| form2.dat | Индикация режима "Выход в ноль" |
| form6.dat | Индикация режима "Выход в точку" |
| form7.dat | Индикация режима "Преднабор" |
| form8.dat | Индикация режима "Фиксированные перемещения" |

### Меню (menu-*.dat)
Файлы с описанием функциональных клавиш.

| Файл | Назначение |
|------|------------|
| menu-te.dat | Меню режимов (F1-F12, токарный вариант) |
| menu-me.dat | Меню модулей (вертикальные клавиши) |

### Модули (modulesm.dat)
Описание встраиваемых модулей (редактор, таблицы, график, осциллограф и др.).

### Таблицы (tables.dat)
Описание таблиц (параметры, инструменты, диагностика).

### Системные файлы
| Файл | Назначение |
|------|------------|
| action.dat | Определение пользовательских команд |
| clock.dat | Элемент "Часы" |
| colors.dat | Цветовая палитра (несколько вариантов) |
| condition.dat | Вывод статуса ЧПУ (Авария/Готово) |
| defaults.dat | Общие константы (шрифты, отступы, константы выравнивания) |
| dialtok.dat | Диалог для токарного варианта (циклы, G-функции, САПП) |
| hotkeys.dat | Назначение горячих клавиш |
| keys.dat | Коды клавиш |
| krd-t.dat | Константы для токарного варианта (признаки осей, координаты форм) |
| m28form.dat | Вывод m28 и m29 (системные сообщения) |
| m28grid.dat | Вывод m28 и m29 в отдельной зоне |
| m_functions.dat | Описание M-функций |
| maya-te.dat | Главный файл подключения (сборка всех компонентов) |

---

## Использование

Файлы могут использоваться для:
1. Изучения синтаксиса на реальных примерах
2. Поиска типовых решений для конкретных задач
3. Тестирования новых конструкций
4. Создания собственных элементов на основе готовых

---

## Примечание

Файлы содержат реальные константы и координаты, специфичные для конкретного проекта. При использовании в других проектах требуется адаптация:
- координаты (BOX_*_X, BOX_*_Y)
- константы шрифтов (FONTSIZE_*, HEIGHT_*)
- константы цветов (B1-B10, MAIN_*, TEXT_*)
- признаки осей (IND_KRD0-IND_KRD5)
- номер шпинделя (IND_S)
