## Математические функции

### abs(выражение)
Абсолютное значение выражения.

Примеры:
x = abs(-10)        ; x = 10
y = abs(d70 - 100)  ; y = |d70 - 100|

---

## Тригонометрические функции

### sin(_rad[_n])
Синус угла.
- _rad — угол в радианах (умноженный на 10^n)
- n — степень увеличения (по умолчанию 3, т.е. в 1000 раз)

Примеры:
a = sin(3141)       ; sin(3.141 рад) * 1000
b = sin(3142, 3)    ; то же самое

### cos(_rad[_n])
Косинус угла.

### tg(_rad[_n])
Тангенс угла.

### asin(_x[_n])
Арксинус.
- _x — аргумент функции (умноженный на 10^n)
- Результат — угол в радианах (умноженный на 10^n)

### acos(_x[_n])
Арккосинус.

### atg(_x[_n])
Арктангенс.

### pi(_[n])
Константа π (3.14159...) с указанной степенью увеличения.
pi() = 3141 (при n=3)
pi(5) = 314159

### pi_2(_[n])
π/2.

### pi_4(_[n])
π/4.

### pi_X(_x, _y[_n])
π * (x / y) * 10^n.

### deg2rad(_a[_n])
Перевод градусов в радианы.
- _a — угол в градусах (умноженный на 10^n)
- Результат — радианы (умноженный на 10^n)

Примеры:
rad = deg2rad(180000)   ; 180° * 1000 → 3141 рад * 1000

## Логические и условные функции

### iif(выражение, выражение2, выражение3)
Вычисляет выражение. Если результат не равен 0, вычисляет и возвращает выражение2, иначе — выражение3.

Пример:
d70 = iif(d38 = 0, 100, 200)   ; если d38=0, то 100, иначе 200

### iif(выражение, выражение2)
Вычисляет выражение. Если результат не равен 0, возвращает 0, иначе — выражение2.

Пример:
d70 = iif(d38 = 0, 1)   ; если d38=0, то 1, иначе 0

### sel(выражение, выражение1, выражение2, ...)
Вычисляет выражение. Если результат равен 1, вычисляет и возвращает выражение1, если равен 2 — выражение2 и т.д. Если нет соответствия — возвращает 0.

Пример:
mode = sel(d32, 100, 200, 300)   ; d32=1 → 100, d32=2 → 200, d32=3 → 300

## Функции работы с подпрограммами

### isrun(номер_пп)
Возвращает 1, если фоновая программа с указанным номером запущена, иначе — 0.

Пример:
if isrun(88) = 0
    run 88          ; запустить, если не запущена
fi

## Функции отслеживания изменений

Эти функции сравнивают текущее значение выражения с предыдущим.
Перед первым вычислением предыдущее значение считается равным 0.
Для корректной работы строка с функцией должна быть выполнена минимум дважды.

### diff(выражение)
Возвращает разницу между текущим и предыдущим значением: Δ = Xn - Xn-1.

Пример:
delta = diff(d70)   ; изменение d70 за такт

### up(выражение)
Возвращает Δ, если Δ > 0, иначе 0.

Пример:
increase = up(d70)  ; положительное изменение

### down(выражение)
Возвращает Δ, если Δ < 0, иначе 0.

Пример:
decrease = down(d70) ; отрицательное изменение

### limit(выражение, граница)
Вычисляет Δ = diff(выражение).
- Возвращает 1, если Δ = 0
- Возвращает 0, если Δ ≠ 0 и граница = 0
- Возвращает 1, если Δ ≠ 0 и граница > 0
- Возвращает 0, если Δ < границы
- Возвращает 1, если Δ ≠ 0 и граница < 0

Пример:
if limit(d70, 5) = 1   ; изменение меньше 5

## Функции работы с памятью

### rsave()
Сохраняет параметры. Необходимо выполнять однократно после внесения изменений в таблицу инструментов из ПЭС.

Пример:
; после изменения T10.3
tsave()
rsave()

### tsave()
Сохраняет таблицу инструментов. Необходимо выполнять однократно.

Пример:
T10.3 = 1000
tsave()

### reset()
Сброс ЧПУ. Аналог нажатия кнопки "Сброс ЧПУ".

Пример:
if A22.0 = 1
    reset()
fi

## Функция ПИД-регулятора

### Pidreg(Kp, Ki, Kd, N, Imax, max, уставка, обратная_связь, сброс, отладка, индекс_координаты)

Выходной сигнал формируется по формуле:
Out = Kp * 10^N * e + Ki * 10^N * ∫edt + Kd * 10^N * de/dt

где:
- e (ошибка) = уставка - обратная_связь
- ∫edt — интеграл ошибки, ограниченный от -Imax до Imax
- de/dt — производная ошибки

| Параметр | Описание |
|----------|----------|
| Kp | Коэффициент пропорциональности |
| Ki | Интегральный коэффициент |
| Kd | Дифференциальный коэффициент |
| N | Порядок коэффициентов (K * 10^N) |
| Imax | Пороговое значение интеграла ошибки |
| max | Пороговое значение выходной величины |
| уставка | Целевое значение |
| обратная_связь | Текущее значение |
| сброс | Признак сброса (1 — сброс) |
| отладка | Признак отладки (1 — вывод в M70-M72) |
| индекс_координаты | Номер координаты (0..9) |

Коэффициенты задаются в виде K * 10^N.
Пример: при Kp=5 и N=-3: Kp*10^N = 5 * 10^-3 = 0.005

Если индекс координаты задан правильно, выходной сигнал поступает в канал поправки на положение соответствующей координаты.

Пример:
out = Pidreg(100, 10, 20, -3, 1000, 32000, d78, m14.0, 0, 1, 0)

## Примеры использования функций

### Пример 1: Абсолютное значение
my delta = abs(d70 - d71)

### Пример 2: Тригонометрия
my angle_deg = 45000      ; 45 градусов * 1000
my angle_rad = deg2rad(angle_deg)
my sin_val = sin(angle_rad)

### Пример 3: Условное присвоение
d40 = iif(d10 = 1, 100, 0)

### Пример 4: Отслеживание изменения скорости
:1
    my speed = diff(d40)
    if speed > 100
        warn "Резкое увеличение скорости"
    fi
    wait
    goto 1

### Пример 5: Ожидание завершения фоновой программы
:10
    run 88
    while isrun(88) = 1
        wait(100)
    wend
    kill

### Пример 6: ПИД-регулятор
:111 RT
    my pid_out = Pidreg(100, 10, 20, -3, 1000, 32000, d78, m14.0, 0, 1, 0)
    ; pid_out автоматически применяется к координате

### Пример 7: Сложное выражение
my result = iif(d38 = 0, 
                abs(d70 - d71), 
                iif(d38 = 1, d70 + d71, 0))

## Особенности использования

1. Тригонометрические функции работают с радианами, умноженными на 1000 (при n=3)
2. Для перевода градусов в радианы используйте deg2rad()
3. Функции diff, up, down, limit запоминают предыдущее значение выражения
4. Строка с этими функциями должна выполняться минимум дважды для корректной работы
5. rsave() и tsave() должны выполняться однократно после изменения данных
6. Pidreg может работать в RT-подпрограммах для непрерывного регулирования
