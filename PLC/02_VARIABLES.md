## Общее описание

В ПЭС используются следующие типы переменных:
- Предопределённые переменные (D, A, R, I, M, P, T)
- Личные переменные (my)
- Машинные переменные (use ... as ...)
- Битовая адресация (переменная.бит)
- Маскированная адресация (переменная & маска)

## D-ячейки

### Синтаксис
Dномер
Dномер[выражение]
D[выражение]

### Диапазон
D0 .. D999

### Назначение
Универсальные ячейки для обмена между БПрО и ПЭС. Могут использоваться для хранения промежуточных данных.

### Привязка к массивам БПрО
В файле hash_d.cfg определяется соответствие D-ячеек элементам массивов БПрО.

### Примеры из hash_d.cfg
D0 i(33) ; Код на ЦАП
D1 p(1) ; Память технологической команды S
D2 p(2) ; Память технологической команды T
D5 i(26) ; Координаты и направление движения при БП от ПЭС
D10 i(3) ; Отработка УП 1-отработка, 0-стоп
D38 m(28) ; Код сообщения
D39 m(29) ; Код ошибки (остановка УП и ПЭС)

### Использование
d10 = 1              ; запуск отработки УП
d38 = 0              ; сброс сообщения
d39 = 251            ; установка кода ошибки

## Входные адреса (A, W)

### Синтаксис
Aномер
Aномер[выражение]
Wномер (синоним A)

### Диапазон
A17 .. A99 (по умолчанию, может меняться в machine.cfg)

### Назначение
Чтение информации с дискретных входов. Младшие 8 бит каждой ячейки соответствуют 8 дискретным входам.

### Примеры
A17.0               ; первый бит адреса 17
A18.3               ; четвёртый бит адреса 18
W17                 ; то же, что A17

### Использование
if A17.0 = 1        ; проверка сигнала
A17.0 = 1           ; принудительная установка (для диагностики)

---

## Выходные адреса (A)

### Синтаксис
Aномер
Aномер[выражение]

### Диапазон
A1 .. A16 (по умолчанию, до A17-1)

### Назначение
Запись информации в дискретные выходы. Младшие 8 бит каждой ячейки соответствуют 8 дискретным выходам.

### Примеры
A1.0 = 1            ; включить первый выход
A2.3 = 0            ; выключить четвёртый выход

## Параметры (R)

### Синтаксис
Rномер
Rномер[выражение]
R[выражение]

### Диапазон
R0 .. R599

### Типы параметров
| Диапазон | Тип | Назначение |
|----------|-----|------------|
| R0..R299 | Одномерные | Коррекции, переменные для УП |
| R300..R399 | Одномерные | Параметры станка (электроавтоматика) |
| R400..R499 | Двумерные (R400.K) | Смещение систем координат (G54-G59) |
| R500..R599 | Двумерные (R500.K) | Геометрические и физические параметры станка |

### Двумерные параметры (с координатой)
R400.0              ; параметр 400 по координате 0 (X)
R500.1              ; параметр 500 по координате 1 (Y)
R400[2]             ; параметр 400 по координате 2 (Z)

### Примеры параметров
R503.0              ; максимальная скорость по X в режиме "Автомат"
R504.0              ; максимальная скорость по X в ручных режимах
R508.0              ; допустимое рассогласование по X
R509.0              ; коэффициент обратной связи по положению (КП) по X
R510.0              ; коэффициент компенсации скоростной ошибки (КСО) по X
R523.0              ; вид работающих координат (биты)

## Массивы БПрО

### I(x) — данные интерполяции (одномерные)
| Индекс | Назначение | Пример |
|--------|------------|--------|
| I(2) | Коррекция скорости подачи 0-200% | i2 |
| I(3) | Команда на запуск отработки УП | i3 |
| I(8) | Скорость безразмерных перемещений от ПЭС | i8 |
| I(13)-I(20) | Минимальные/максимальные скорости шпинделя по диапазонам | i13, i14 |
| I(32) | Вращение шпинделя (0-нет,1-плюс,2-минус) | i32 |
| I(48) | Общая коррекция скорости ручных перемещений | i48 |

### I(x.K) — данные интерполяции (двумерные)
| Индекс | Назначение | Пример |
|--------|------------|--------|
| I(1.K) | Скорость фиксированного перемещения по координате | i1.0 |
| I(3.K) | Величина фиксированного перемещения по координате | i3.0 |
| I(6.M) | Множитель маховика M | i6.0 |
| I(7.M) | Указатель координаты маховика M | i7.0 |

### M(x) — данные индикации (одномерные)
| Индекс | Назначение | Пример |
|--------|------------|--------|
| M(0) | Текущий режим работы УЧПУ | m0 |
| M(1) | Номер управляющей программы | m1 |
| M(2) | Номер кадра управляющей программы | m2 |
| M(5) | Номер активного инструмента | m5 |
| M(20) | Состояние отработки УП | m20 |
| M(28) | Код сообщения | m28 |
| M(29) | Код ошибки | m29 |
| M(30) | Готовность УЧПУ (0-нет,255-есть) | m30 |

### M(x.K) — данные станка (двумерные)
| Индекс | Назначение | Пример |
|--------|------------|--------|
| M(0.K) | Рассогласование по координате | m0.0 |
| M(1.K) | Текущее (расчетное) значение координаты | m1.0 |
| M(2.K) | Остаток пути по координате | m2.0 |
| M(14.K) | Реальное (измеренное) значение координаты | m14.0 |
| M(21.K) | Код на ЦАП по координате | m21.0 |

### P(x) — данные из УП
| Индекс | Назначение | Пример |
|--------|------------|--------|
| P(1) | Действующее значение S (скорость шпинделя) | p1 |
| P(2) | Действующее значение T (номер инструмента) | p2 |
| P(3) | Команда на сброс УЧПУ от ПЭС | p3 |
| P(6) | Команда на смену режима работы УЧПУ | p6 |
| P(8) | Текущий номер инструмента при работе с ТИ | p8 |
| P(13) | Выход в 0 в положительном направлении (от ПЭС) | p13 |
| P(14) | Выход в 0 в отрицательном направлении (от ПЭС) | p14 |

## Таблица инструментов (T)

### Синтаксис
Tномер.индекс
Tномер[индекс]

### Диапазон
T0 .. T199 (номера инструментов)
Индекс 0..9 (значения в элементе)

### Назначение
Хранение данных по инструментам (длина, радиус, коррекции и др.)

### Примеры
T10.3               ; третье значение десятого элемента таблицы инструментов
T[10][3]            ; то же самое

## Личные переменные (my)

### Синтаксис
my имя_переменной
my имя_переменной = значение
my имя1, имя2, имя3
my имя1 = значение1, имя2, имя3 = значение3

### Глобальные переменные
Объявляются после %% и до первой подпрограммы. Доступны во всех подпрограммах.

### Локальные переменные
Объявляются внутри подпрограммы. Доступны только в этой подпрограмме. При каждом вызове подпрограммы создаются заново.

### Примеры
my counter = 0
my flag, status
my speed = 100, position = d78
my Память_скорости = d40

---

## Машинные переменные (use ... as ...)

### Синтаксис
use переменная as псевдоним
use переменная1 as псевдоним1, переменная2 as псевдоним2

### Глобальные переменные
Объявляются после %% и до первой подпрограммы.

### Локальные переменные
Объявляются внутри подпрограммы.

### Примеры
use m10[2] as markerZ
use m1 as UPno
use p0 as zero_param
use A21.2 as Avar

### Битовая переменная
use a1.1 as avar_bit1
avar_bit1 = 1   ; установить первый бит a1
avar_bit1 = 0   ; сбросить первый бит a1

### Маскированная переменная
use a1 & 12 as avar_masked   ; маска 12 (биты 2 и 3)
avar_masked = 15              ; установятся только биты 2 и 3

## Битовая адресация

### Синтаксис
переменная.номер_бита

### Нумерация битов
Биты нумеруются с 0 (младший) по 31 (старший)

### Примеры
A17.0               ; первый бит входного адреса 17
D100.3              ; третий бит D-ячейки 100
m30.1               ; первый бит переменной m30
p7.0                ; нулевой бит p7
my_flag.2           ; второй бит личной переменной

### Использование
A17.0 = 1           ; установить бит
A17.0 = 0           ; сбросить бит
if A17.0 = 1        ; проверить бит
if my_flag.2 = 0    ; проверить бит личной переменной

---

## Маскированная адресация

### Синтаксис
переменная & маска

### Описание
При присвоении значения изменяются только биты, соответствующие единичным битам маски. Остальные биты остаются без изменений.

### Примеры
D123 & 6 = 255      ; установить биты 1 и 2 в 1 (6 = 00000110b)
D123 & 6 = 0        ; сбросить биты 1 и 2
D23 & 1 = 1         ; установить бит 0
D23 & 2 = 0         ; сбросить бит 1

### С использованием переменной
my ZX = 5           ; 5 = 00000101b (биты 0 и 2)
my flag
flag & ZX = 15      ; установить биты 0 и 2 по маске 5

---

## Присвоение группе переменных

### Синтаксис
переменная_начало, количество = значение

### Описание
Присваивает значение указанному количеству последовательных переменных.

### Примеры
D1, 10 = 0         ; очищает D1, D2, ..., D10
D1.4, 10 = 1       ; устанавливает четвертый бит D1, D2, ..., D10
D1&3, 10 = 0       ; очищает биты 0 и 1 D1, D2, ..., D10

### С использованием выражений
my ddd = 5
D1, ddd + ddd = 0  ; очищает D1, D2, ..., D10

## Примеры использования всех типов переменных

%%d

; глобальные переменные
my g_timeout = 1000
use A17.0 as start_btn
use A18.0 as stop_btn
use A1.0 as spindle_on

:1
    ; проверка входов
    if start_btn = 1
        d10 = 1      ; запуск УП
        d49 = 1      ; включение шпинделя
        g_timeout = 1000
    fi
    
    if stop_btn = 1
        d10 = 0      ; останов УП
        d49 = 0      ; останов шпинделя
    fi
    
    ; проверка готовности
    if m30 = 255
        spindle_on = 1   ; включить выход
    else
        spindle_on = 0   ; выключить выход
    fi
    
    ; чтение параметров
    if d40 > R504.0
        d40 = R504.0     ; ограничение скорости
    fi
    
    wait(100)
    goto 1
