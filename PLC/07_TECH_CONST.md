## Общее описание

Технологические константы (ТК) определяют, как БПрО обрабатывает M, S, T-функции, встреченные в управляющей программе. Каждая ТК задаётся в начале ПЭС после указания кодировки.

### Синтаксис
N номер_функции P значение_ТК

---

## Номера функций

| Диапазон | Тип | Описание |
|----------|-----|----------|
| 0-99 | M0-M99 | Вспомогательные функции |
| 100 | S | Скорость шпинделя |
| 101 | T | Инструмент |
| 120-899 | M120-M899 | Дополнительные M-функции |

### Обязательные функции
Следующие функции должны быть обязательно описаны:
- N0 Pxxxxx — M0 (программный останов)
- N1 Pxxxxx — M1 (технологический останов)
- N2 Pxxxxx — M2 (конец программы)
- N30 Pxxxxx — M30 (конец программы с перемоткой)

---

## Структура значения ТК

Значение ТК — пятиразрядное число: P Y5 Y4 Y3 Y2 Y1

### Разряд Y2Y1 (группа)
| Значение | Описание |
|----------|----------|
| 1-15 | Номер группы |

Технологические функции, которые необходимо отрабатывать одновременно в одном кадре, должны относиться к разным группам.

### Разряд Y3 (индикация)
| Значение | Описание |
|----------|----------|
| 0 | Функция индицируется до прихода другой функции из той же группы |
| 1 | Функция индицируется только на время отработки подпрограммы |

Индикация производится в ячейки M12-M15.

### Разряд Y4 (синхронизация с геометрией)
| Значение | Описание |
|----------|----------|
| 0 | Подпрограмма запускается одновременно с геометрией, переход к следующему кадру после выхода из подпрограммы |
| 1 | Подпрограмма запускается до геометрии, ожидается выход, затем отрабатываются перемещения |
| 2 | Подпрограмма запускается после геометрии, переход к следующему кадру после выхода из подпрограммы |
| 3 | Подпрограмма запускается одновременно с геометрией без ожидания окончания |

### Разряд Y5 (наличие перемещений)
| Значение | Описание |
|----------|----------|
| 0 | В функции нет перемещений |
| 1 | В функции есть перемещения от ПЭС |

## Примеры технологических констант

### M0, M1, M2, M30 (обязательные)
N0 P02101    ; M0, группа 1, индикация до смены, после геометрии, без перемещений
N1 P02101    ; M1, группа 1, индикация до смены, после геометрии, без перемещений
N2 P02101    ; M2, группа 1, индикация до смены, после геометрии, без перемещений
N30 P02101   ; M30, группа 1, индикация до смены, после геометрии, без перемещений

### M3, M4, M5 (включение/выключение шпинделя)
N3 P1005     ; M3, группа 5, индикация на время, до геометрии, без перемещений
N4 P1005     ; M4, группа 5, индикация на время, до геометрии, без перемещений
N5 P1105     ; M5, группа 5, индикация на время, после геометрии, без перемещений

### M8, M9 (СОЖ)
N8 P1004     ; M8, группа 4, индикация на время, до геометрии, без перемещений
N9 P1004     ; M9, группа 4, индикация на время, до геометрии, без перемещений

### M41-M44 (диапазоны шпинделя)
N41 P1004    ; M41, группа 4, индикация на время, до геометрии, без перемещений
N42 P1004    ; M42, группа 4, индикация на время, до геометрии, без перемещений
N43 P1004    ; M43, группа 4, индикация на время, до геометрии, без перемещений
N44 P1004    ; M44, группа 4, индикация на время, до геометрии, без перемещений

### M6 (смена инструмента)
N6 P0202     ; M6, группа 2, индикация до смены, после геометрии, с перемещениями

### S-функция
N100 P1005   ; S, группа 5, индикация на время, до геометрии, без перемещений

### T-функция
N101 P1005   ; T, группа 5, индикация на время, до геометрии, без перемещений

## Правила задания технологических констант

1. Технологические константы должны располагаться после строки с кодировкой (%%) и до первой подпрограммы

2. Для M0, M1, M2, M30 обязательно должны быть заданы ТК

3. M17 не требует ТК (специальная функция)

4. Для S и T номер группы не имеет значения, но должен быть указан в пределах 1-15

5. Функции, которые должны выполняться одновременно, должны иметь разные номера групп

6. Если функция имеет перемещения от ПЭС (Y5=1), БПрО учитывает это при подготовке кадра

## Часто используемые конфигурации ТК

### Типовая конфигурация для M-функций
P1005 — M-функция, группа 5, индикация на время, до геометрии, без перемещений

### Для функций, изменяющих состояние
P1004 — M-функция, группа 4, индикация на время, до геометрии, без перемещений

### Для функций с перемещениями
P0202 — M-функция, группа 2, индикация до смены, после геометрии, с перемещениями

### Для останова
P02101 — M0/M1/M2/M30, группа 1, индикация до смены, после геометрии, без перемещений

---

## Особые случаи

### M17
Не требует ТК. Вызывается автоматически для отработки функции G17 (выбор плоскости XY).

### M100-M119
Зарезервированы для системных функций, не рекомендуется использовать.

### M120-M899
Могут использоваться для пользовательских функций.

### Отмена MST-функций
Параметр R194 позволяет отменить отработку:
- R194 & 1 — отмена M
- R194 & 2 — отмена S
- R194 & 4 — отмена T
