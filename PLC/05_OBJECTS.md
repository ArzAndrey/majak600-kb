## Общее описание

Объекты в ПЭС — это структуры данных с методами, которые позволяют реализовать:
- Счётчики
- Вычисление выражений
- Сигналы (детекторы фронта)
- Отслеживание изменений
- Таймеры (импульсы, задержки)
- Обмен данными по CAN/RS-485

Создание объекта:
my имя_объекта = тип_объекта(параметры)

Вызов метода:
имя_объекта::метод
имя_объекта::метод(аргументы)

## Объект counter (счётчик)

### Синтаксис
my объект = counter(конец)
my объект = counter(начало, конец)
my объект = counter(начало, конец, шаг)

### Описание
Счётчик с шагом. Начальное значение устанавливается на начало-шаг. Первым должен вызываться метод next.

### Методы
| Метод | Описание |
|-------|----------|
| next() | Увеличить на шаг, вернуть 1 если не исчерпан, иначе 0 |
| prev() | Уменьшить на шаг, вернуть 1 если успешно, иначе 0 |
| val() | Вернуть текущее значение |
| reset() | Сброс в начальное состояние |
| reset(начало, конец, шаг) | Перенастройка и сброс |

### Примеры
my o = counter(0, 10, 2)
my z
while o::next
    z = o::val    ; значения: 0, 2, 4, 6, 8, 10
wend

my o2 = counter(5)    ; от 0 до 5 с шагом 1

my o3 = counter(1, 10) ; от 1 до 10 с шагом 1

## Объект expr (вычисление выражений)

### Синтаксис
my объект = expr(выражение1[, выражение2, выражение3, ...])

### Описание
Хранит список выражений и вычисляет их по запросу.

### Методы
| Метод | Описание |
|-------|----------|
| calc() | Вычислить первое выражение и вернуть результат |
| calc(n) | Вычислить выражение n (1-основное) |

### Примеры
my x = 10, z
my o = expr(x, x*x, x+x)

z = o::calc      ; z = 10
z = o::calc(0)   ; z = 0
z = o::calc(1)   ; z = 10
z = o::calc(2)   ; z = 100 (10*10)
z = o::calc(3)   ; z = 20 (10+10)

## Объект sig (сигнал, детектор фронта)

### Синтаксис
my объект = sig(-1, выражение)   ; детектор отрицательного фронта (1→0)
my объект = sig(0, выражение)    ; детектор любого фронта (0→1, 1→0)
my объект = sig(1, выражение)    ; детектор положительного фронта (0→1)

### Описание
Отслеживает изменение выражения и выдает сигнал при обнаружении фронта.

### Методы
| Метод | Описание |
|-------|----------|
| check() | Проверить выражение, вернуть 1 при обнаружении фронта |

### Примеры
my x, z
my o = sig(-1, x)   ; отрицательный фронт

z = o::check ; z=0
x = 1
z = o::check ; z=0
x = 0
z = o::check ; z=1
x = 0
z = o::check ; z=0

my o2 = sig(1, x)   ; положительный фронт
x = 1
z = o2::check ; z=1
x = 1
z = o2::check ; z=0

## Объект delta (отслеживание изменения)

### Синтаксис
my объект = delta(выражение)

### Описание
Отслеживает изменение значения выражения между вызовами.

### Методы
| Метод | Описание |
|-------|----------|
| diff() | Вычислить разницу между текущим и предыдущим значением |
| limit(граница) | Вернуть результат функции limit() |
| up() | Вернуть 1 при увеличении значения |
| down() | Вернуть 1 при уменьшении значения |

### Примеры
my x = 10
my o = delta(x)

z = o::diff    ; z=0 (первый вызов)
x = 5
z = o::diff    ; z=-5
x = 10
z = o::diff    ; z=5

; использование limit
if o::limit(3) = 1
    ; изменение менее 3
fi

; использование up/down
if o::up = 1
    ; значение увеличилось
fi

## Объект pulse (генератор импульсов)

### Синтаксис
my объект = pulse(выражение_вход, время_t1)
my объект = pulse(выражение_вход, время_t1, время_t2)

### Описание
Формирует импульс заданной длительности. При наличии t2 формирует меандр.

### Методы
| Метод | Описание |
|-------|----------|
| reset() | Сброс в исходное состояние |
| out() | Выходной сигнал (0 или 1) |

### Режимы работы
- Без t2: сигнал на выходе = 1, пока таймер t1 активен И есть сигнал на входе
- С t2: чередование t1 (выход=1) и t2 (выход=0) пока есть сигнал на входе

### Примеры
; импульс длительностью 0.4 сек при нажатии кнопки
my pulse1 = pulse(A17.0, 400)
A1.0 = pulse1::out

; меандр с периодом 1 сек (0.4 вкл, 0.6 выкл)
my pulse2 = pulse(A17.0, 400, 600)
A1.0 = pulse2::out

## Объект xpulse (удлинённый импульс)

### Синтаксис
my объект = xpulse(выражение_вход, время)

### Описание
Формирует импульс длительностью время. При повторном срабатывании до истечения времени таймер перезапускается.

### Методы
| Метод | Описание |
|-------|----------|
| reset() | Сброс в исходное состояние |
| out() | Выходной сигнал (0 или 1) |

### Примеры
; удлинённый импульс 2 сек
my xp = xpulse(A17.0, 2000)
A1.0 = xp::out

## Объект dev (обмен по CAN/RS-485)

### Синтаксис
my объект = dev("имя_устройства", индекс, подиндекс, длина)

### Описание
Обеспечивает обмен данными с устройством, подключенным по CAN или Modbus. Работает только в RT-подпрограммах.

### Параметры
| Параметр | Описание |
|----------|----------|
| имя_устройства | Имя из конфигуратора оборудования |
| индекс | Индекс объекта в устройстве |
| подиндекс | Подиндекс объекта |
| длина | Длина данных в байтах |

### Методы
| Метод | Описание |
|-------|----------|
| write(значение) | Записать данные |
| read() | Прочитать данные |
| get() | Получить считанное значение |
| status() | Состояние (0-ожидание, 1-выполнено, 2-ошибка) |
| geterr() | Код ошибки |

### Примеры
:111 RT
    my drive = dev("LENZE-A", 24521, 0, 4)   ; ток двигателя
    
    drive::read
    wait if drive::status = 0
    if drive::status = 1
        current = drive::get / 1000
    fi

## Объект pidreg (ПИД-регулятор)

### Синтаксис
my объект = pidreg(Kp, Ki, Kd, N, Imax, max, уставка, обратная_связь, сброс, отладка, индекс_координаты)

### Описание
Аналог функции Pidreg, но в виде объекта.

### Методы
| Метод | Описание |
|-------|----------|
| pidout() | Получить выходное значение |
| reset(параметры) | Сброс с новыми параметрами |

### Примеры
my reg = pidreg(100, 10, 20, -3, 1000, 32000, d78, m14.0, 0, 1, 0)
out = reg::pidout
reg::reset(200, 20, 40, -3, 1000, 32000, d78, m14.0, 0, 1, 0)

## Полные примеры использования объектов

### Пример 1: Счётчик для цикла
:1
    my i = counter(0, 10)
    while i::next
        d70 = i::val
        wait(100)
    wend
    kill

### Пример 2: Детектор фронта кнопки
:1
    my btn = sig(1, A17.0)   ; положительный фронт
    
    if btn::check = 1
        d10 = 1 - d10        ; инвертировать
    fi
    wait
    goto 1

### Пример 3: Отслеживание скорости
:1
    my speed_delta = delta(d40)
    
    if speed_delta::limit(100) = 0
        warn "Скорость изменилась более чем на 100"
    fi
    wait
    goto 1

### Пример 4: Мигающий индикатор
:1 RT
    my blink = pulse(1, 500, 500)   ; меандр 0.5/0.5 сек
    A1.0 = blink::out

### Пример 5: Защита от дребезга
:1
    my debounce = on(A17.0, 50)     ; 50 мс задержка включения
    my signal = debounce::out
    A1.0 = signal

### Пример 6: Чтение данных с привода
:111 RT
    my drive = dev("DRIVE_X", 16#606C, 0, 4)   ; скорость
    my count = 0
    
    while count < 10
        drive::read
        wait if drive::status = 0
        if drive::status = 1
            speed[count] = drive::get
            count = count + 1
        fi
    wend
