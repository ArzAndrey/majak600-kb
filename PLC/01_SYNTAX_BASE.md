## Общее описание

ПЭС (Программа Электроавтоматики Станка) — это программа, написанная на интерпретируемом языке, которая выполняется на УЧПУ "Маяк-600" параллельно с Базовым Программным Обеспечением (БПрО). ПЭС обеспечивает привязку БПрО к конкретному станку, реализует обработку дискретных сигналов, управление вспомогательными механизмами, обработку M,S,T-функций и другие специфические функции.

## Технологические константы

Технологические константы определяют поведение M,S,T-функций.

### Синтаксис
N номер_функции P значение_ТК

### Номера функций
| Номер | Тип |
|-------|-----|
| 0-899 | M-функции |
| 100 | S-функция (скорость шпинделя) |
| 101 | T-функция (инструмент) |

### Структура значения ТК (P Y5 Y4 Y3 Y2 Y1)
| Разряд | Значение | Описание |
|--------|----------|----------|
| Y2Y1 | 1..15 | Номер группы (одновременная отработка в одном кадре — разные группы) |
| Y3 | 0 или 1 | Индикация: 0 — до смены, 1 — только на время отработки |
| Y4 | 0,1,2,3 | Синхронизация с геометрией: 0 — одновременно, 1 — до геометрии, 2 — после геометрии, 3 — одновременно без ожидания |
| Y5 | 0 или 1 | Наличие перемещений в функции |

### Примеры
N0 P02101    ; M0, группа 1, индикация до смены, после геометрии, без перемещений
N1 P02101    ; M1, аналогично
N2 P02101    ; M2
N30 P02101   ; M30
N8 P1004     ; M8, группа 4, индикация только на время отработки, до геометрии
N100 P1005   ; S, группа 5, индикация только на время отработки, до геометрии

## Переменные

### Предопределённые переменные
| Тип | Синтаксис | Диапазон | Описание |
|-----|-----------|----------|----------|
| D-ячейки | Dxxx | D0..D999 | Универсальные ячейки обмена |
| Входные адреса | Axxx | A17..A99 | Дискретные входы |
| Выходные адреса | Axxx | A1..A16 | Дискретные выходы |

| Параметры | Rxxx | R0..R599 | Параметры станка |

| Массив M | M(x) или M(x.K) | | Данные индикации и станка |

### Примеры

D100[24]
D24[200-100]

## Битовая адресация

### Синтаксис
переменная.номер_бита

### Нумерация битов
Биты нумеруются с 0 (младший) по 31 (старший)

### Примеры
A17.0          ; первый бит входного адреса 17
D100.3         ; третий бит D-ячейки 100
m30.1          ; первый бит переменной m30
p7.0           ; нулевой бит p7

### Использование
A17.0 = 1      ; установить бит
A17.0 = 0      ; сбросить бит
if A17.0 = 1   ; проверить бит

## Маскированная адресация

### Синтаксис
переменная & маска

### Описание
При присвоении значения изменяются только биты, соответствующие единичным битам маски. Остальные биты остаются без изменений.

### Примеры
D123 & 6 = 255   ; установить биты 1 и 2 в 1 (6 = 00000110b)
D123 & 6 = 0     ; сбросить биты 1 и 2

### С использованием переменной
my ZX = 5        ; 5 = 00000101b
my flag
flag & ZX = 15   ; установить биты 0 и 2 по маске 5

## Системы счисления

### Синтаксис
число[d]      — десятичная (по умолчанию)
число[b]      — двоичная
число[o]      — восьмеричная
число[h]      — шестнадцатеричная

### Примеры
1234           ; десятичное 1234
1234d          ; десятичное 1234
11101011b      ; двоичное 235
353o           ; восьмеричное 235
1A34h          ; шестнадцатеричное 6708
0FFFh          ; шестнадцатеричное 4095 (обязательно с ведущим нулём)

### Десятичная точка
число.число    ; умножается на 1000

### Примеры
0.001          ; = 1
1.0            ; = 1000
2.14           ; = 2140
123            ; = 123000

## Личные переменные (my)

### Синтаксис
my имя_переменной
my имя_переменной = значение
my список_переменных

### Глобальные переменные
Объявляются после %% и до первой подпрограммы. Доступны во всех подпрограммах.

### Локальные переменные
Объявляются внутри подпрограммы. Доступны только в этой подпрограмме. При каждом вызове подпрограммы создаются заново.

### Примеры
my var = 10
my var2, var3
my var3 = var + var2
my Память_скорости

## Машинные переменные (use ... as ...)

### Синтаксис
use переменная as псевдоним
use список_переменных

### Глобальные переменные
Объявляются после %% и до первой подпрограммы.

### Локальные переменные
Объявляются внутри подпрограммы.

### Примеры
use m10[2] as markerZ
use m1 as UPno
use p0 as zero_param
use A21.2 as Avar
use a1.1 as avar_bit1
use a1 & 12 as avar_masked

### Битовая переменная
use a1.1 as avar_bit1
avar_bit1 = 1   ; установить первый бит a1

### Маскированная переменная
use a1 & 12 as avar_masked   ; маска 12 (биты 2 и 3)
avar_masked = 15              ; установятся только биты 2 и 3

## Арифметические выражения

### Операторы (в порядке приоритета)
| Приоритет | Операторы | Описание |
|-----------|-----------|----------|
| 1 | - (унарный) | Изменение знака |
| 2 | * / % | Умножение, деление, остаток |
| 3 | + - | Сложение, вычитание |
| 4 | << >> | Сдвиг влево/вправо |
| 5 | < > <= >= | Сравнение |
| 6 | = != | Равенство, неравенство |
| 7 | & | Побитовое И |
| 8 | # | Побитовое исключающее ИЛИ |
| 9 | | | Побитовое ИЛИ |
| 10 | && | Логическое И |
| 11 | || | Логическое ИЛИ |

### Приоритет можно изменять с помощью круглых скобок

### Примеры
x = 10
y = 3
a = x + y      ; 13
b = x * y      ; 30
c = x / y      ; 3
d = x % y      ; 1
e = x & y      ; 2 (1010b & 0011b = 0010b)
f = x | y      ; 11 (1010b | 0011b = 1011b)
g = x # y      ; 9 (1010b # 0011b = 1001b)
h = x << y     ; 80 (1010b << 3 = 1010000b)
i = x >> y     ; 1 (1010b >> 3 = 0001b)

### Цепочки сравнений
if (x <= y < z < w)    ; эквивалентно if (x <= y && y < z && z < w)
if (0 != X <= 10)      ; эквивалентно if ((X != 0) && (X <= 10))

## Пример минимального файла ПЭС

%%d
; технологические константы
N0 P02101
N1 P02101
N2 P02101
N30 P02101
N100 P1005

; глобальные переменные
my g_counter = 0
use A17.0 as start_button

; подпрограмма 1 (фоновая)
:1
    ; опрос кнопки пуска
    if start_button = 1
        d10 = 1
    fi
    wait
    goto 1

; подпрограмма 100 (M-функция)
:100 P1005
    ; включение шпинделя
    d49 = 1
    wait(100)
    kill
