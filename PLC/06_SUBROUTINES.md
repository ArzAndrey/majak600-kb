## Общее описание

Подпрограммы (ПП) — это основные структурные единицы ПЭС. Каждая подпрограмма имеет номер и может быть вызвана из БПрО или из других подпрограмм.

### Синтаксис
:номер_пп [тип] [технологическая_константа]
    операторы
    ...

### Подпрограммы M-функций (100-199)
Вызываются при появлении в УП M-функций.

| Диапазон | Назначение |
|----------|------------|
| 100-199 | M0-M99 (100 соответствует M0, 101-M1, ..., 199-M99) |

### Системные подпрограммы БПрО (200-219)
Вызываются БПрО для выполнения специальных функций.

| № ПП | Назначение |
|------|------------|
| 200 | Отработка S (передаёт управление ПП9) |
| 201 | Отработка T (передаёт управление ПП10) |
| 202 | Отработка G28 |
| 203 | Запуск ПП 205-211 (выход в ноль) |
| 204 | Запуск счётчиков в D96-D103 |
| 205-211 | Выход в ноль по координатам |
| 212 | Запуск дополнительных фоновых программ |
| 216 | Быстрая автоматика (опрос аварийных КВ) |
| 218 | Фиксированные перемещения, выход в точку, выход в ноль |
| 219 | Корневая ПЭС |

### Подпрограммы пользователя (220-999)
Могут использоваться для любых целей.

## Типы подпрограмм

### main (главная)
Подпрограмма, вызываемая сразу после старта ПЭС.

Синтаксис:
:номер main

Пример:
:219 main
    ; инициализация
    d10 = 0
    d49 = 0
    run 1
    kill

### RT (реального времени)
Подпрограмма, вызываемая с заданным периодом.

Синтаксис:
:номер RT
:номер RT(период)

Период указывается в миллисекундах:
- RT или RT(0) — каждый такт (2.5 мс при 400 Гц)
- RT(5) — каждые 5 мс (каждый второй такт)
- RT(1000) — каждую секунду

Ограничения RT-подпрограмм:
- Нельзя использовать глобальные личные переменные (my после %%)
- Нельзя использовать косвенную адресацию (Dxxx[выражение])
- Нельзя использовать call и run с меткой

Пример:
:216 RT(5)    ; быстрая автоматика, период 5 мс
    ; опрос аварийных КВ
    if A17.0 = 0
        d10 = 0
        d49 = 0
    fi

## Технологические константы

Для подпрограмм 100-199 можно указать технологическую константу.

Синтаксис:
:номер Pзначение

Структура значения: P Y5 Y4 Y3 Y2 Y1

| Разряд | Значение | Описание |
|--------|----------|----------|
| Y2Y1 | 1..15 | Номер группы |
| Y3 | 0 или 1 | Индикация (0-до смены, 1-только на время) |
| Y4 | 0,1,2,3 | Синхронизация (0-одновременно, 1-до геометрии, 2-после, 3-без ожидания) |
| Y5 | 0 или 1 | Наличие перемещений |

Примеры:
:100 P1005    ; M0, группа 5, индикация на время, до геометрии
:103 P1005    ; M3, группа 5, индикация на время, до геометрии
:105 P1105    ; M5, группа 5, индикация на время, после геометрии
:141 P1004    ; M41, группа 4, индикация на время, до геометрии

## Вызов подпрограмм

### call — вызов с возвратом
call номер_пп
call номер_пп : метка
call номер_пп if условие

Вызванная подпрограмма выполняется, после kill управление возвращается к следующему оператору.

### run — запуск в фоне
run номер_пп
run номер_пп : метка
run номер_пп if условие

Подпрограмма запускается параллельно. Не ждёт завершения.

### kill — останов
kill              ; завершить текущую подпрограмму
kill номер_пп     ; остановить фоновую подпрограмму

## Метки

Метка — это имя, заканчивающееся символом ":".

Синтаксис:
имя:

Использование:
goto имя
call 100 : имя
wait имя

Пример:
:100
    start:
        d10 = 1
        wait(1000)
        d10 = 0
        goto start

## Примеры подпрограмм

### Пример 1: Фоновая программа (ПП1)
:1
    ; опрос кнопок пульта
    if A17.0 = 1
        d10 = 1
    fi
    
    if A18.0 = 1
        d10 = 0
    fi
    
    wait(100)
    goto 1

### Пример 2: Инициализация при включении (ПП23)
:23
    ; установка начальных значений
    d10 = 0
    d40 = 100
    d49 = 0
    d38 = 0
    d39 = 0
    
    ; установка выходов
    A1.0 = 0
    A2.0 = 0
    
    ; запуск фоновых программ
    run 1
    run 5
    
    kill

### Пример 3: Обработка M-функции (M3 - ПП103)
:103 P1005
    ; включение шпинделя
    d49 = 1
    
    ; ожидание готовности привода
    while A20.0 = 0
        wait(50)
    wend
    
    ; ожидание выхода на скорость
    wait(500)
    
    ; проверка ошибок
    if A21.0 = 1
        error "Ошибка привода шпинделя"
        d49 = 0
        kill
    fi
    
    kill

### Пример 4: Обработка S-функции (ПП9)
:9
    ; выбор диапазона по скорости
    if d1 < d50
        d63 = 1          ; диапазон 1
        call 41
    elif d1 <= d51
        d63 = 1          ; диапазон 1
    elif d1 <= d53
        d63 = 2          ; диапазон 2
        call 42
    elif d1 <= d55
        d63 = 3          ; диапазон 3
        call 43
    else
        d63 = 5
        error "Скорость вне диапазона"
    fi
    
    kill

### Пример 5: Выход в ноль по координате X (ПП205)
:205
    ; установка параметров перемещения
    d78 = 1000           ; величина перемещения
    d40 = R525.0         ; скорость поиска зоны останова
    d47.0 = 1            ; направление "+X"
    
    ; движение до датчика
    while A17.0 = 0
        wait
    wend
    
    ; съезд с датчика
    d40 = R526.0         ; вторая скорость
    while A17.0 = 1
        wait
    wend
    
    ; повторный наезд
    d40 = R527.0         ; третья скорость
    while A17.0 = 0
        wait
    wend
    
    ; установка признака готовности
    d319.0 = 1
    
    kill

### Пример 6: Фоновая программа с запуском по условию
:26
    ; запускается при D34 & 1 != 0
    ; дополнительная обработка
    while 1
        ; специальные функции
        wait(1000)
    wend

### Пример 7: RT-подпрограмма быстрой автоматики
:216 RT(5)
    ; аварийный останов
    if A19.0 = 0          ; аварийный выключатель
        d10 = 0
        d49 = 0
        A1.0 = 0
        A2.0 = 0
        error "Аварийный останов"
    fi
    
    ; аварийные КВ
    if A17.0 = 0
        d47.0 = 0
        d10 = 0
        error "Сработал аварийный КВ X"
    fi

### Пример 8: Запуск фонового поиска инструмента
:10
    ; обработка T-функции
    if d2 != p8
        run 88           ; запустить поиск инструмента в фоне
    fi
    kill

:88
    ; поиск инструмента
    ; ... (длительная операция)
    kill

:106
    ; M6 - смена инструмента
    while isrun(88) = 1  ; ждём завершения поиска
        wait(100)
    wend
    ; смена инструмента
    kill
