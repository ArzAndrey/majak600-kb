## Правильный синтаксис (только так!)

| Конструкция | Правильно | Неправильно |
|-------------|-----------|-------------|
| Форма | `{ form 0 ... } form` | `form 0 ... endform` |
| Вид | `{ view ... } view` | `view ... endview` |
| Элемент | `{ item ... }` | `item ...` |
| Меню | `{ menu 10 { f1 ... } } menu` | `menu 0 ... horizontal` |
| Условие | `{ if m30=255 ... }` | `if m30=255 ... endif` |
| Цвет | `bgr = MAIN_BGR` | `bgcolor = black` |
| Координата | `x = GAP_H` | `x = 10` |

## Пример правильного кода

{ default xmax=800 ymax=600 fgr=MAIN_TEXT bgr=MAIN_BGR } default

{ form 0 (menu=10, xmax=665, ymax=600)
    { view ()
        { item # (height=FONTSIZE_MID) GAP_H GAP_V yes yes 'Автомат' }
        { item m30 (invbgr=Error_C invfgr=MAIN_BGR width=106)
            111 GAP_V yes yes 'ЧПУ: Авария'
            { if m30=255 invbgr=GREEN format='ЧПУ: Готово' }
        }
    } view
} form

# ⛔ ЗАПРЕЩЕННЫЕ КОНСТРУКЦИИ (НЕ ИСПОЛЬЗОВАТЬ!)

| Неправильно (НЕ РАБОТАЕТ) | Правильно (РАБОТАЕТ) |
|---------------------------|---------------------|
| `form 0 "АВТОМАТ"` | `{ form 0 ... } form` |
| `size 800 600` | `xmax=800, ymax=600` |
| `bgcolor MAIN_BGR` | `bgr = MAIN_BGR` |
| `{ grid rows 7 cols 1 }` | `{ grid g_main (grid_rows=7, grid_cols=1) } grid` |
| `type text` | `{ item ... }` |
| `if ... else ... endif` | `{ if ... }` в конце item |
| `{ f1 "ПУСК" }` | `{ f1 { text "ПУСК" } { pressed ... } }` |
| `xoffset=200` | Только абсолютные координаты `x` |

**ВСЕГДА используй синтаксис из файлов базы знаний!**

## 1. HMI — язык разметки интерфейса

### Базовый синтаксис
{ form N (params) ... } form          ; корневой экран (N: 0-Авт,1-Ручн,2-Выход0,6-ВыходТочка,7-Преднаб,8-ФП)
{ grid name (params) ... } grid       ; сетка
{ view (params) ... } view            ; область
{ item name (params) X Y inv always 'text' } ; элемент
{ box color=RGB x= X y= Y x2= W y2= H } ; залитый прямоугольник
{ rect color=RGB x= X y= Y x2= W y2= H } ; рамка
{ line color=RGB x= X y= Y x2= X2 y2= Y2 } ; линия
{ menu N { f1...f14 } } menu          ; меню
{ default params } default            ; параметры по умолчанию

### Размеры экрана
800x600  (10.4") или 1024x768 (15")
Начало координат: (0,0) — левый верхний угол

### Цвета (из colors.dat)
MAIN_BGR      ; основной фон (B7)
MAIN_TEXT     ; основной текст (B1)
Error_C       ; ошибка (RED)
GOT_C         ; готовность (GREEN)
Messag_C      ; сообщение (YELLOW)
AN_C          ; текст в режимах Автомат/Наладка
MENU_BGR      ; фон меню
MENU_FGR      ; текст меню

### Константы (из defaults.dat)
GAP_H = 19            ; горизонтальный отступ
GAP_V = 16            ; вертикальный отступ
FONTSIZE_MID = 20     ; средний шрифт
FONTSIZE_SMALL = 14   ; малый шрифт
FONTSIZE_BIG = 19     ; большой шрифт
HEIGHT_MID = 17       ; высота строки
BOX_X = 506           ; координата X блоков
BOX_W = 142           ; ширина блоков

### Форматы вывода
[5]         ; 5 знаков с нулями слева
[+7]        ; с плюсом/минусом
[9.3]       ; 9 знаков, 3 после точки
[+9.3]      ; с плюсом/минусом и точкой
[6!2]       ; отсечь 2 знака справа
&(d70*2)[]  ; вычисляемое выражение
<51>        ; текст из txtvals.dat

### Условия
{ if m30=255 invbgr=GREEN format='Готово' }
{ if (d70>0)&&(d70<100) fgr=YELLOW }

### Команды в item
+inc#stop -dec#stop    ; в item
; в action.dat:
{ action inc d70 = d70 + 1 }
{ action dec d70 = d70 - 1 }

### Команды в меню
pressed mm10           ; переключить меню
pressed grid_name      ; вызвать сетку
pressed d10=1          ; установить D-ячейку
pressed %=d404#0 ifok 3 "1" start_log 0  ; условная команда

## 2. ПЭС — язык электроавтоматики

### Структура файла
%%d                      ; кодировка (dos/koi8/win)
N0 P02101                ; технологические константы
N1 P02101
...
my var = 0               ; глобальные переменные
use A17.0 as start_btn   ; машинные переменные

:1                       ; подпрограмма 1 (фоновая)
    операторы
    wait
    goto 1

:103 P1005               ; M3 с технологической константой
    операторы
    kill

### Типы переменных
D0-D999         ; D-ячейки
A1-A16          ; выходные адреса
A17-A99         ; входные адреса
R0-R399         ; одномерные параметры
R400.0-R599.9   ; двумерные параметры (с координатой)
I(0)-I(94)      ; данные интерполяции
M(0)-M(108)     ; данные индикации
M(0.K)-M(55.K)  ; данные станка (с координатой)
P(0)-P(31)      ; данные из УП
T0.0-T199.9     ; таблица инструментов
my var           ; личные переменные
use var as name  ; машинные переменные

### Битовая адресация
A17.0 = 1        ; установить бит
A17.0 = 0        ; сбросить бит
if A17.0 = 1     ; проверить бит

### Маскированная адресация
D123 & 6 = 255   ; установить биты 1 и 2
D23 & 1 = 1      ; установить бит 0

### Операторы
var = expr               ; присвоение
call 100                 ; вызов ПП с возвратом
run 88                   ; запуск фоновой ПП
kill                     ; завершить текущую ПП
kill 88                  ; остановить фоновую ПП
wait                     ; задержка на такт
wait(100)                ; задержка 100 мс
wait if A17.0 = 1        ; ожидание условия
wait(1000) if m30=255    ; ожидание условия или таймаута
goto label               ; переход на метку
if d10=1 ... fi          ; условие
if d10=1 ... else ... fi
while d40>0 ... wend     ; цикл
msg "Текст"              ; сообщение
warn "Текст"             ; предупреждение
error "Ошибка"           ; ошибка

### Функции
abs(x)                   ; абсолютное значение
iif(cond, v1, v2)        ; условное значение
isrun(88)                ; проверка запуска фоновой ПП
sin(3141)                ; синус (угол*1000)
cos(3141)                ; косинус
deg2rad(45000)           ; перевод градусов в радианы
diff(expr)               ; изменение значения
up(expr)                 ; положительное изменение
down(expr)               ; отрицательное изменение
limit(expr, граница)     ; проверка изменения
rsave()                  ; сохранение параметров
tsave()                  ; сохранение таблицы инструментов
reset()                  ; сброс ЧПУ

### Объекты
my cnt = counter(0,10,2) ; счётчик
cnt::next, cnt::val      ; методы

my sig = sig(1, A17.0)   ; детектор фронта
if sig::check = 1        ; проверка

my delta = delta(d40)    ; отслеживание изменения
delta::diff, delta::up, delta::down

my pls = pulse(A17.0, 400) ; генератор импульсов
pls::out                 ; выход

my on_delay = on(A17.0, 1000) ; задержка включения
my off_delay = off(A17.0, 1000) ; задержка выключения

### Технологические константы
N0 P02101    ; M0, группа 1, индикация до смены, после геометрии
N1 P02101    ; M1
N2 P02101    ; M2
N30 P02101   ; M30
N3 P1005     ; M3, группа 5, индикация на время, до геометрии
N4 P1005     ; M4
N5 P1105     ; M5, после геометрии
N8 P1004     ; M8, группа 4
N100 P1005   ; S-функция
N101 P1005   ; T-функция

## 3. Ключевые D-ячейки

| D-ячейка | Назначение |
|----------|------------|
| d10 | Запуск УП (1) / Стоп (0) |
| d11 | Режим (0-Авт,1-Ручн,2-Вых0,6-ВыхТ,7-Преднаб,8-ФП) |
| d38 | Код сообщения |
| d39 | Код ошибки |
| d40-d46 | Скорость перемещения по координатам |
| d47 | Направление "+" (бит) |
| d48 | Направление "-" (бит) |
| d49 | Шпиндель (1-вперёд,2-назад,0-стоп) |
| d78-d84 | Величина перемещения (мкм) |
| m0 | Текущий режим |
| m1 | Номер УП |
| m2 | Номер кадра |
| m5 | Номер инструмента |
| m20 | Состояние отработки УП (6-отработка) |
| m28 | Код сообщения |
| m29 | Код ошибки |
| m30 | Готовность (255-есть) |

## 4. Частые шаблоны

### HMI: форма с одним видом
{ default xmax=800 ymax=600 fgr=MAIN_TEXT bgr=MAIN_BGR } default
{ form 0 (menu=10, xmax=665, ymax=600)
    { view ()
        { item # () GAP_H GAP_V yes yes 'Автомат' }
    } view
} form

### HMI: элемент с условием
{ item m30 (invbgr=Error_C invfgr=MAIN_BGR width=106)
    111 GAP_V yes yes 'ЧПУ: Авария'
    { if m30=255 invbgr=GREEN format='ЧПУ: Готово' }
}

### HMI: элемент с командой
; в action.dat:
{ action inc d40 = d40 + 10 }
{ action dec d40 = d40 - 10 }
; в форме:
{ item d40 (height=FONTSIZE_MID width=80)
    GAP_H GAP_V no yes 'F [4]' +inc -dec }

### ПЭС: фоновая программа
:1
    if A17.0 = 1
        d10 = 1
    fi
    if A18.0 = 1
        d10 = 0
        d49 = 0
    fi
    wait(100)
    goto 1

### ПЭС: M-функция
:103 P1005
    d49 = 1
    kill

:104 P1005
    d49 = 2
    kill

:105 P1105
    d49 = 0
    kill

### ПЭС: фиксированное перемещение
d78 = 10000    ; 10 мм
d40 = 500      ; 500 мм/мин
d47.0 = 1      ; направление "+X"
while d40 != 0
    wait
wend

### ПЭС: выход в ноль
d78 = 1000
d40 = R525.0
d47.0 = 1
while A21.0 = 0
    wait
wend
d40 = R526.0
while A21.0 = 1
    wait
wend
d319.0 = 1

### ПЭС: ожидание фоновой программы
run 88
while isrun(88) = 1
    wait(100)
wend

### ПЭС: сообщение об ошибке
if A19.0 = 0
    error "Аварийный останов"
    d10 = 0
    d49 = 0
    A1.0 = 0
fi
